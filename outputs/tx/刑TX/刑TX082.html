❀A-1 問題文

正当防衛に関する次の記述ア（1）からオ（5）までの各記述を判例の立場に従って検討した場合、正しいものはどれか。各記述に ○（結論・あてはめが正しい）／×（誤り）を付して検討する。

前【前提】── 正当防衛（刑法36条1項）成否を判例の立場で判定する

各記述は、いずれも甲の反撃行為について正当防衛が成立するかを問う独立した事例である。判例が要求する①急迫不正の侵害（侵害が現に存在しているか間近に押し迫っていること）、②侵害が自己又は他人の権利に対するものであること、③防衛の意思に基づくこと、④やむを得ずにした行為（相当性）の各要件への当てはめが正しいかで、記述の正誤が決まる。記述の「正当防衛が成立する／成立しない」という結論と、判例の正しい当てはめとが一致するものが ○ となる。

【記述】── 正当防衛の成否を問う5事例（ア〜オ＝肢1〜5）

ア甲は、乙が甲所有の自動車を盗むのを目撃し、これを追跡したものの見失い、その翌日、窃取された場所から約2キロメートル離れた路上で、乙がその自動車から降りて立ち去ったのを認めた。甲は、乙がすぐに戻って来る様子であったので、直ちにその自動車を運転し、自宅に戻った。この場合、甲には正当防衛が成立する。
イ甲は、散歩中、仲の悪かった乙から大型犬をけしかけられたので、犬から逃げようとして、偶然その場を通り掛かった丙を突き飛ばして走り去った。甲の行為により、丙は転倒して全治約1週間を要する足首捻挫の傷害を負った。この場合、甲には正当防衛が成立する。
ウ甲は、乙ら数名の男によって監禁されたが、監禁されて2週間後、たまたま見張りが乙一人になったので、監禁場所から脱出するため、乙の顔面を1回殴打して乙がひるんだ隙にそこから逃げた。この場合、甲には正当防衛が成立する。
エ甲は、深夜、路上で、見知らぬ乙から、ナイフを胸元に突き付けられ現金を要求されたので、ナイフを避けるために乙の胸付近を手で押し、走って逃げ出した。甲の行為により、乙は転倒して後頭部を路面に打ち付け、全治約1か月間を要する頭部打撲の傷害を負った。この場合、甲には正当防衛は成立しない。
オ甲は、同居していた乙と言い争いをし、乙から「ぶっ殺すぞ。」と怒鳴られたため、身の危険を感じて一旦家を出たが、乙と仲直りをしようと考え直し、乙から暴力を振るわれることがあるかもしれないと思いつつ、家に戻って乙に謝罪した。しかし、甲は、乙に数回顔面を殴られた上、更に殴り続けられそうになったことから憤激し、とっさに乙の脇腹付近を1回蹴り、乙に全治約1か月間を要する肋骨骨折の傷害を負わせた。この場合、甲には正当防衛は成立しない。

❀A-2 解答

各記述の正誤を ○×で判定

記述ア〜オ（肢1〜5）それぞれに ○（正しい）／×（誤り）を付けてから「解答を表示」を押してください（導き書§3-1：必ず5記述全部に○×を付けてから reveal）。正解の肢番号は○×から自動で定まります。

ア 翌日2km先で取り戻した自動車運転に正当防衛が成立する、とする
イ 犬から逃げる際に第三者・丙を突き飛ばす行為に正当防衛が成立する、とする
ウ 監禁（継続犯）からの脱出時、見張りの乙を1回殴打する行為に正当防衛が成立する、とする
エ ナイフ強盗の胸を手で押した行為に正当防衛は成立しない、とする
オ 暴力を予期しつつ戻り、殴られて憤激し1回蹴った行為に正当防衛は成立しない、とする

正解　3　（記述ウのみ正しい）
正しい肢を1つ選ぶのではなく、各記述に現れた正当防衛の要件あてはめで正誤を判定する。下表の「論点のコア」が学習資産。

記述正誤登場した論点のコア（転用可能な法理）
ア×窃盗は状態犯。窃取が既遂・終了した翌日の取戻しは「過去の侵害」で急迫性を欠く（最判昭46.11.16）。正当防衛は不成立で、結論「成立する」は誤り。
イ×正当防衛は侵害者に向けた反撃を正当化する制度。侵害者でない第三者・丙への加害は正当防衛では正当化されない。結論「成立する」は誤り。
ウ○監禁は継続犯。監禁中は不正の侵害が現在し急迫性あり。脱出目的で防衛の意思あり、1回殴打は必要相当。正当防衛が成立し結論は正しい。
エ×反撃が必要最小限度で相当なら、生じた結果がたまたま大でも防衛行為でなくならない（最判昭44.12.4）。胸を押す行為は相当で成立。結論「成立しない」は誤り。
オ×侵害をあらかじめ予期しても直ちに急迫性を失わない。憤激・逆上のみでは防衛の意思を欠かず、積極的加害意思の特別事情もない（最判昭46.11.16）。成立し結論「成立しない」は誤り。

ア
✗ 誤（窃盗は状態犯で既遂後の取戻しは「過去の侵害」＝急迫性を欠く）
記述アは、翌日約2km先で乙が降りて立ち去った自動車を甲が運転して取り戻した行為に正当防衛が成立する、とする。しかし窃盗は状態犯で、窃取が既遂・終了した翌日の取戻しは過去の侵害に対するもの。急迫性を欠き正当防衛は成立しないため、結論「成立する」は誤り。
  1. 急迫性の定義：最判昭46.11.16は、刑法36条の「急迫」を法益侵害が現に存在するか間近に押し迫っていることと定義する。過去の侵害・過ぎ去った侵害には防衛行為はできない。
  2. 状態犯としての窃盗：窃盗罪は状態犯であり、財物の占有を取得した時点で既遂・終了する。窃取後に違法状態が続いても、侵害行為そのものは終了している。
  3. ひっかけの型：「自救行為」と「正当防衛」の混同。侵害がすでに終了した後の取戻しは急迫性を欠く。乙が戻りそうでも、新たな侵害が間近に迫った事実はない。
記述原文

ア（1）　甲は、乙が甲所有の自動車を盗むのを目撃し、これを追跡したものの見失い、その翌日、窃取された場所から約2キロメートル離れた路上で、乙がその自動車から降りて立ち去ったのを認めた。甲は、乙がすぐに戻って来る様子であったので、直ちにその自動車を運転し、自宅に戻った。この場合、甲には正当防衛が成立する。

📖 解説原文

正当防衛（刑法36条1項）が成立するには、まず急迫不正の侵害が必要である。最判昭46.11.16によれば、「急迫」とは法益の侵害が現に存在しているか、又は間近に押し迫っていることを意味し、この定義から、過去の侵害・過ぎ去った侵害には防衛行為はできないことが導かれる。本記述では、甲は乙による窃盗の際に追跡したが見失い、翌日に乙が同自動車から降りて立ち去ったのを認めている。窃盗罪は状態犯であるから、乙が甲の自動車を盗んだ時点で犯罪が成立し、犯罪成立と同時に終了する。したがって、甲が乙に盗まれた自動車を運転した時点では、乙による窃盗行為は過去の侵害といえ、法益の侵害が現に存在し又は間近に押し迫っているとはいえない。よって「急迫」の要件を欠くため、甲が自動車を運転し自宅に持ち去ったことについて正当防衛は成立しない。結論「成立する」とする記述アは誤りである。

📚 根拠条文・判例

→ 刑法36条1項（正当防衛・急迫性） → 最判昭46.11.16（「急迫」の意義） → 刑法235条（窃盗罪＝状態犯）

👨‍🏫 教授の解説

1ポイント

要するに、侵害が「過去のもの」になっていないかを問うています。正当防衛は刑法36条の「急迫」、すなわち侵害が現在しているか間近に迫っていることを要する。窃盗が状態犯で既遂後に終了している以上、翌日の取戻しは過去の侵害への対応であり急迫性を欠く──この時間軸の把握が決め手。

2考え方の道筋

「急迫」の時間的判定：最判昭46.11.16の「現に存在しているか、又は間近に押し迫っている」の定義に当てはめる。窃盗（状態犯）は占有取得で既遂・終了するから、翌日の場面で侵害は終わっており、急迫性の要件で正当防衛は脱落する。

記述アのズレの所在：本問は取り戻しという「自救行為」を正当防衛に擬している。乙がすぐ戻りそうという事情も、新たな侵害が間近に迫った事実を作るものではない。過去の侵害に対する実力行使は正当防衛の枠外であり、結論「成立する」は誤り。

イ
✗ 誤（侵害者でない第三者・丙への加害は正当防衛では正当化されない）
記述イは、乙にけしかけられた犬から逃げる際、偶然通り掛かった第三者・丙を突き飛ばして負傷させた行為に正当防衛が成立する、とする。しかし正当防衛は侵害者に対する反撃を正当化する制度であり、侵害者以外の第三者に生じた法益侵害を正当化するものではない。結論「成立する」は誤り。
  1. 正当防衛の対抗関係：刑法36条1項の正当防衛は、急迫不正の侵害をなす者（侵害者）に対する反撃を正当化する制度。反撃の相手方は侵害者でなければならない。
  2. 第三者への加害の処理：侵害者でない第三者に生じた法益侵害は、正当防衛では正当化されない。問題となり得るのは緊急避難（37条）であり、正当防衛と枠組みが異なる。
  3. ひっかけの型：「正当防衛の根拠は侵害者の法益の質的劣位性（不正対正の関係）にある」。無関係な丙にはこの劣位性が及ばず、正対正の関係になるため正当防衛は使えない。
記述原文

イ（2）　甲は、散歩中、仲の悪かった乙から大型犬をけしかけられたので、犬から逃げようとして、偶然その場を通り掛かった丙を突き飛ばして走り去った。甲の行為により、丙は転倒して全治約1週間を要する足首捻挫の傷害を負った。この場合、甲には正当防衛が成立する。

📖 解説原文

正当防衛は、侵害者に対する侵害を正当化するものであり、侵害者以外の第三者に生じた法益侵害を正当化するものではないと解されている。その理由として、学説は、正当防衛は「急迫不正の侵害」者の法益に対する、被侵害者の法益の質的な優位性を違法性阻却の根拠とするものであるという点を挙げている。本記述では、甲は、偶然その場を通り掛かった、侵害者以外の第三者である丙を突き飛ばしているため、この甲の行為について正当防衛は成立しない。なお、丙への加害は緊急避難（刑法37条）の成否が別途問題となり得るにとどまる。結論「成立する」とする記述イは誤りである。

📚 根拠条文・判例

→ 刑法36条1項（正当防衛・対侵害者） → 刑法37条1項（緊急避難・第三者への加害）

👨‍🏫 教授の解説

1ポイント

要するに、反撃の相手方が「侵害者」か「無関係な第三者」かを問うています。正当防衛は不正の侵害をなす者に対してのみ使える。突き飛ばされた丙は侵害者ではないため、丙への加害は正当防衛では正当化されない──反撃の向き先の確認が決め手。

2考え方の道筋

違法性阻却の根拠からの帰結：正当防衛が広く正当化されるのは、相手が「不正」の侵害者であり、その法益が被侵害者の法益に対して質的に劣位するからである。この劣位性は侵害者にのみ妥当する。

記述イのズレの所在：丙は侵害者でない無関係の第三者であり、甲との関係は「正対正」になる。質的劣位性が成り立たない以上、正当防衛の枠組みは使えない。残るのは緊急避難（37条）の問題であって、正当防衛が成立するとする記述イは的を外している。

👨‍🏫 教授の解説

1ポイント

要するに、監禁中は侵害が「現在」しているかを問うています。監禁罪は継続犯だから、監禁が続く間ずっと不正の侵害が現在し、急迫性が認められる。脱出目的（防衛の意思）も、1回殴打という相当性も満たすため、正当防衛が成立する──犯罪類型（継続犯）の理解が決め手。

2考え方の道筋

4要件への当てはめ：①急迫不正の侵害＝監禁（継続犯）が現在進行中で侵害が現在、②自己の権利（身体の自由）、③防衛の意思＝脱出目的、④相当性＝2週間監禁という重大な法益侵害に対し顔面1回殴打は必要最小限度。すべて充足する。

記述アとの対比：同じ最判昭46.11.16の「急迫」定義でも、状態犯（窃盗・記述ア）は既遂で侵害終了、継続犯（監禁・記述ウ）は違法状態が侵害として継続という違いが結論を分ける。両記述の対比で「急迫性」の判定を体得するとよい。

最判昭44.12.4は、刑法36条1項にいう「已むことを得ざるに出てたる行為」（やむを得ずにした行為）とは、急迫不正の侵害に対する反撃行為が、自己又は他人の権利を防衛する手段として必要最小限度のものであること、すなわち反撃行為が侵害に対する防衛手段として相当性を有するものであることを意味するのであって、反撃行為が右の限度を超えず、したがって侵害に対する防衛手段として相当性を有する以上、その反撃行為により生じた結果がたまたま侵害されようとした法益より大であっても、その反撃行為が正当防衛行為でなくなるものではないと解すべきである、としている。本記述では、ナイフという凶器を胸元に突きつけた上で現金を要求するという乙の侵害行為は甲の身体の安全への高度の危険性を有する行為であるのに対して、甲の行為は乙の胸付近を手で押したという軽微なものにすぎない。そうだとすれば、結果として乙は全治約1か月を要する傷害を負っているものの、甲の反撃行為は自己の権利を防衛する手段として必要最小限度のものといえ、相当性の範囲内にあり「やむを得ずにした行為」といえる。よって、甲には正当防衛が成立する。結論「成立しない」とする記述エは誤りである。

📚 根拠条文・判例

→ 刑法36条1項（正当防衛・相当性） → 最判昭44.12.4（相当性＝行為の必要最小限度）

👨‍🏫 教授の解説

1ポイント

要するに、相当性は「行為」で判定するのか「結果の大小」で判定するのかを問うています。最判昭44.12.4は反撃行為が必要最小限度かで相当性を判断し、結果がたまたま大きくても防衛行為でなくならないとする。ナイフ強盗に手で押すのは相当──判定対象の理解が決め手。

2考え方の道筋

相当性の判定方法：侵害の危険性（ナイフを胸元に突き付ける強盗＝身体への高度の危険）と反撃手段（胸付近を手で押す＝軽微）を比較する。反撃は明らかに必要最小限度で相当性の範囲内にある。

記述エのズレの所在：記述エは「全治1か月」という結果の重大さから逆算して相当性を否定している。しかし判例は行為の相当性で判断し、結果の大小は相当性を左右しない。結果に引きずられた誤った当てはめで、結論「成立しない」は誤り。

記述オは、暴力を振るわれるかもしれないと思いつつ家に戻った甲が、乙に殴られて憤激し脇腹を1回蹴った行為について、正当防衛は成立しない、とする。しかし侵害があらかじめ予期されていても直ちに急迫性を失わず、憤激・逆上のみでは防衛の意思を欠かない（最判昭46.11.16）。積極的加害意思の特別事情もなく、正当防衛は成立し、結論「成立しない」は誤り。
  1. 予期された侵害と急迫性：最判昭46.11.16は、侵害があらかじめ予期されていたとしても、そのことから直ちに急迫性を失わないとする。急迫性には法益侵害の客観的な切迫性があれば足りる。
  2. 憤激・逆上と防衛の意思：相手の加害行為に対し憤激・逆上して反撃しても、直ちに防衛の意思を欠くものではない。攻撃を受けたのに乗じ積極的加害意思に出たなどの特別事情がない限り、防衛の意思は認められる。
  3. ひっかけの型：「暴力を予期して戻った」「憤激して蹴った」という事情に引きずられた急迫性・防衛意思の安易な否定。予期や憤激だけでは正当防衛を排除しないのが判例。
記述原文

オ（5）　甲は、同居していた乙と言い争いをし、乙から「ぶっ殺すぞ。」と怒鳴られたため、身の危険を感じて一旦家を出たが、乙と仲直りをしようと考え直し、乙から暴力を振るわれることがあるかもしれないと思いつつ、家に戻って乙に謝罪した。しかし、甲は、乙に数回顔面を殴られた上、更に殴り続けられそうになったことから憤激し、とっさに乙の脇腹付近を1回蹴り、乙に全治約1か月間を要する肋骨骨折の傷害を負わせた。この場合、甲には正当防衛は成立しない。

📖 解説原文

最判昭46.11.16は、「急迫」とは法益の侵害が現に存在しているか、又は間近に押し迫っていることを意味し、その侵害があらかじめ予期されていたものであるとしても、そのことからただちに急迫性を失うものと解すべきではないとしている。よって、甲が事前に乙による暴力を予想していたことをもって「急迫」性を欠くことにはならない。また、判例は、刑法36条の防衛行為は防衛の意思をもってなされることが必要であるが、相手の加害行為に対し憤激又は逆上して反撃を加えたからといって、ただちに防衛の意思を欠くものと解すべきでないとした上で、「攻撃を受けたのに乗じ積極的な加害行為に出たなどの特別な事情が認められないかぎり、被告人の反撃行為は防衛の意思をもってなされたものと認めるのが相当である」としている。本記述では、甲は乙から更に殴り続けられそうになったことに対応するために攻撃を加えており、乙から攻撃を受けたのに乗じ積極的な加害行為に出たなどの特別な事情も認められないから、防衛の意思が認められる。また、甲は顔面を殴打されるという乙の侵害行為に対して、乙の脇腹付近を1回蹴ったにすぎず、甲の行為は「やむを得ずにした行為」といえる。よって、甲には正当防衛が成立する。結論「成立しない」とする記述オは誤りである。

📚 根拠条文・判例

→ 刑法36条1項（正当防衛・急迫性／防衛の意思） → 最判昭46.11.16（予期された侵害・防衛の意思）

👨‍🏫 教授の解説

1ポイント

要するに、「侵害の予期」と「憤激」が正当防衛を排除するかを問うています。最判昭46.11.16は、侵害を予期しても直ちに急迫性を失わず、憤激・逆上のみでは防衛の意思を欠かないとする。積極的加害意思の特別事情がない本問は成立する──二つの判例命題の理解が決め手。

2考え方の道筋

急迫性の検討：暴力を予期して戻ったとしても、判例上それだけでは急迫性を失わない。現に乙が殴り続けようとした時点で、法益侵害は客観的に切迫しており急迫性が認められる。

防衛の意思の検討：憤激して蹴ったという事情があっても、判例は憤激・逆上のみでは防衛の意思を否定しない。攻撃に乗じた積極的加害意思などの特別事情がない以上、防衛の意思は認められる。脇腹を1回蹴った行為も相当性の範囲内であり、正当防衛が成立する。結論「成立しない」は誤り。
