前本問の共通前提 ── 「乙の承諾があると誤信した甲」を5罪で検討

各記述は、いずれも甲が「乙の承諾がない（または承諾の前提を欠く）のに、これがあると誤信して」行為に及んだ場面である。被害者の承諾（同意）が①そもそも違法阻却・構成要件該当性阻却に意味を持つか、②持つとして甲の承諾に関する錯誤が故意を阻却するか、という二段の検討を、罪ごとに当てはめる。承諾が無意味な罪・違法な目的ゆえ承諾が無効な罪では、承諾の誤信があっても犯罪は成立する。

ア甲は、乙にわいせつな行為をすることについての乙の承諾がないのに、これがあると誤信して、乙が10歳であることを知りながら、乙に対してわいせつな行為を行った。（13歳未満の者に対する不同意わいせつ罪）
イ甲は、乙を殺害することについての乙の承諾がないのに、これがあると誤信して、乙の首をひもで絞めて殺害した。（同意殺人罪）

ア
✓ 正（13歳未満わいせつ罪は承諾が無意味＝承諾の誤信も無意味で成立する）
乙が10歳である以上、不同意わいせつ罪（176条3項）は本人の同意があっても成立する。承諾は犯罪成立上そもそも無意味であるから、甲が乙の承諾を誤信したことも結論を左右せず、甲には13歳未満の者に対する不同意わいせつ罪が成立する。
  1. 13歳未満の絶対的保護：刑法176条3項は、客体が13歳未満の場合、本人の同意があっても不同意わいせつ罪を成立させる。性的自己決定能力の未成熟ゆえ承諾に意味を認めない絶対的保護の趣旨。
  2. 承諾の無意味＝錯誤も無意味：承諾自体が犯罪成否に影響しない罪では、承諾の存否に関する錯誤も故意・違法性を左右しない。年齢の認識（10歳と知っていた）があれば構成要件的故意に欠けない。
  3. 承諾論の入口：被害者の承諾はまず「その罪で承諾が意味を持つか」を判定する。意味を持たない罪では、承諾の有無・その誤信を論じる前に犯罪が成立する。
記述原文

ア　甲は、乙にわいせつな行為をすることについての乙の承諾がないのに、これがあると誤信して、乙が10歳であることを知りながら、乙に対してわいせつな行為を行った。（13歳未満の者に対する不同意わいせつ罪）

📖 解説原文

被害者の承諾が犯罪の成否に影響を及ぼすのは、その承諾が保護法益の処分として意味を持つ場合に限られる。不同意わいせつ罪（刑法176条3項）は、13歳未満の者が客体である場合、本人の同意があっても成立する。これは13歳未満の者の性的自己決定能力の未成熟に鑑み、その同意に法的意味を認めず絶対的に保護する趣旨である。したがって13歳未満の者による同意は犯罪成立において意味がない。本記述では、甲が10歳の乙の承諾があると誤信していたことは何ら意味がなく、しかも甲は乙が10歳であることを知っていたのであるから、客体の年齢に関する故意にも欠けるところはない。よって甲には13歳未満の者に対する不同意わいせつ罪が成立する。

📚 根拠条文・判例

→ 刑法176条3項（13歳未満への不同意わいせつ） → 被害者の承諾の体系（承諾が意味を持つ罪・持たない罪）

👨‍🏫 教授の解説

1ポイント

要するに、13歳未満の者に対するわいせつ罪で「承諾」は意味を持つのかを問うています。176条3項は13歳未満なら同意があっても成立させる絶対的保護の罪。承諾が無意味である以上、その承諾を誤信したことも結論に影響しない──ここを押さえれば即決できる。

2考え方の道筋

承諾の意味の有無を先に判定：被害者の承諾論は、まず「その罪・その被害者で承諾が法益処分として意味を持つか」を判定する。13歳未満わいせつでは性的自己決定能力の未成熟を理由に承諾に意味を認めない。よって承諾の存否も、それを誤信したか否かも犯罪成否を左右しない。

故意の検討：甲は乙が10歳であることを知っていた。客体の年齢という構成要件要素の認識があるから、構成要件的故意に欠けない。承諾の誤信は故意阻却事由として機能する余地がなく、不同意わいせつ罪が成立する。

✓ 正（承諾を誤信した殺害は抽象的事実の錯誤で軽い同意殺人罪が成立）
乙の承諾がないのに承諾ありと誤信して殺害した場合、重い殺人罪（199条）の客観面に、軽い同意殺人罪（202条）の認識という抽象的事実の錯誤が生じる。両罪の構成要件が重なる範囲で軽い同意殺人罪が成立する（名古屋地判平7.6.6）。
  1. 抽象的事実の錯誤の処理：認識（同意殺人）と実現（普通殺人）が異なる構成要件にまたがる場合、両罪の構成要件が実質的に重なり合う範囲で、軽い罪（同意殺人罪）の故意既遂を認める（法定的符合説・構成要件的符合）。
  2. 殺人罪と同意殺人罪の包摂関係：202条の同意殺人は199条の殺人の減軽類型で、保護法益（生命）も行為態様も重なり合う。軽い同意殺人の限度で符合する。
  3. 承諾の誤信は有利に作用：承諾なきを承諾ありと誤信した者は、より軽い罪を意図していた。重い結果の故意を問えず、認識した軽い罪で処断するのが責任主義に適う。
記述原文

イ　甲は、乙を殺害することについての乙の承諾がないのに、これがあると誤信して、乙の首をひもで絞めて殺害した。（同意殺人罪）

📖 解説原文

名古屋地判平7.6.6は、「被告人は、被害者Aの嘱託がないのにこれがあるものと誤信して殺害行為に及んだことが明らかであるから……嘱託殺人罪の罪責を負う」と判示した。すなわち、被害者の真意に基づく同意はないのに、同意があるものと誤信して殺害した場合には、抽象的事実の錯誤として、重い殺人罪（199条）の客観面に軽い同意殺人罪（202条）の主観面が対応する。両罪の構成要件は生命という保護法益において重なり合うから、その重なる範囲で軽い同意殺人罪が成立する。甲は承諾があると誤信していた以上、より軽い同意殺人の故意しか認められず、重い普通殺人の故意は問えない。よって甲には同意殺人罪が成立する。

📚 根拠条文・判例

→ 刑法199条（殺人罪） → 刑法202条（同意殺人罪） → 名古屋地判平7.6.6（嘱託の誤信＝同意殺人）

👨‍🏫 教授の解説

1ポイント

要するに、承諾がないのに承諾ありと誤信して殺した者を、殺人罪と同意殺人罪のどちらで処断するかを問うています。認識（軽い同意殺人）と実現（重い普通殺人）が食い違う抽象的事実の錯誤。重なる範囲＝軽い同意殺人罪で処断する、というのが結論です。

2考え方の道筋

錯誤の型を特定：甲は「承諾ある殺害（＝同意殺人）」を認識し、「承諾なき殺害（＝普通殺人）」を実現した。異なる構成要件にまたがる抽象的事実の錯誤である。法定的符合説では、両構成要件が実質的に重なり合う範囲で軽い罪の故意既遂を認める。

符合の範囲：199条の殺人と202条の同意殺人は、生命侵害という点で重なり合い、202条は199条の減軽類型として包摂関係に立つ。重なるのは軽い同意殺人の限度。承諾ありと誤信した甲には、その認識に対応する同意殺人罪が成立する（名古屋地判平7.6.6と同旨）。

✗ 誤（承諾の不存在は構成要件該当事実＝その誤信は故意を阻却し不成立）
最判昭58.4.8は、住居侵入罪の「侵入」を管理権者の意思に反する立入りと解し、承諾の不存在を構成要件該当事実とする。承諾ありと誤信した甲には構成要件該当事実の錯誤があり、故意が阻却されるから住居侵入罪は成立しない。
  1. 「侵入」＝意思侵害（意思侵害説）：最判昭58.4.8は130条前段の「侵入」を管理権者の意思に反する立入りと解する。承諾の不存在は構成要件該当事実そのもの。
  2. 違法性阻却ではなく構成要件レベル：承諾は違法性阻却事由として後から効くのではなく、承諾の不存在が「侵入」の成立要件に組み込まれる。だから承諾の誤信は構成要件的事実の錯誤として故意を阻却する。
  3. 承諾の体系的位置で結論が変わる：承諾が構成要件該当性を否定する罪では、その不存在の誤信が故意を阻却する。承諾が無意味・無効な罪（ア・エ）とは扱いが逆になる。
記述原文

ウ　甲は、乙の居宅に入ることについての乙の承諾がないのに、これがあると誤信して、乙が単身居住する乙の居宅に入った。（住居侵入罪）

📖 解説原文

最判昭58.4.8（百選Ⅱ16事件）は、「刑法130条前段にいう『侵入シ』とは、他人の看守する建造物等に管理権者の意思に反して立ち入ることをいうと解するべきである」として、承諾の不存在を構成要件該当事実であるとしている。したがって、本記述のように、乙の居宅に入ることについて乙の承諾がないのに、これがあると誤信して居宅に入った甲には、構成要件該当事実に関する事実の錯誤があり、故意が阻却される。承諾の存在は違法性阻却事由として後から評価されるのではなく、「侵入」という構成要件要素そのものに承諾の不存在が組み込まれている点が決め手である。よって甲に住居侵入罪は成立しない。

📚 根拠条文・判例

→ 刑法130条前段（住居侵入罪） → 最判昭58.4.8（「侵入」＝意思侵害説） → 被害者の承諾の体系（構成要件阻却型）

👨‍🏫 教授の解説

1ポイント

要するに、住居侵入罪で「承諾」はどの段階で効くのかを問うています。最判昭58.4.8の意思侵害説では、承諾の不存在が「侵入」という構成要件要素そのもの。だから承諾ありと誤信すれば構成要件的事実の錯誤として故意が阻却され、不成立になります。

2考え方の道筋

承諾の体系的位置を確定：承諾が違法性阻却事由なら、その誤信は違法性の錯誤や誤想行為の問題になる。だが住居侵入罪では「侵入＝意思に反する立入り」と定義され、承諾の不存在が構成要件に内在する。承諾の有無は構成要件該当性のレベルで効く。

錯誤の効果：構成要件該当事実（＝承諾の不存在）を認識していなければ、構成要件的故意が欠ける。承諾ありと誤信した甲は「意思に反する立入り」という事実の認識を欠くから、故意が阻却され住居侵入罪は成立しない。承諾の体系的位置づけ次第で錯誤の効果が逆転する好例。

✓ 正（保険金詐取という違法な目的に利用するための承諾は違法性を阻却せず傷害罪が成立）
最決昭55.11.13は、傷害について承諾があっても、保険金を騙取する違法な目的に利用するために得られた承諾は違法性を阻却しないとする。承諾が傷害罪の違法性を阻却しない以上、甲がその承諾を誤信していても傷害罪が成立する。
  1. 承諾の有効性は諸般の事情で判断：最決昭55.11.13は、傷害罪の成否は単に承諾の存在だけでなく、承諾を得た動機・目的・手段・方法・傷害の部位程度等を照らし合わせて決すべきとする。
  2. 違法な目的に利用する承諾＝無効：保険金詐取という違法な目的に利用するために得た承諾は、当該傷害行為の違法性を阻却しない。承諾は有効に成立していない。
  3. 承諾無効型＝誤信も無意味：そもそも違法性を阻却しない承諾だから、それを誤信したか否かも結論を左右しない。承諾が無効な罪では承諾の誤信は故意を阻却しない。
記述原文

エ　甲は、乙に傷害を負わせることについての乙の承諾がないのに、これがあると誤信して、過失による事故を装って保険金を詐取するため、甲の運転する自動車を乙に衝突させ、乙に傷害を負わせた。（傷害罪）

📖 解説原文

最決昭55.11.13（百選Ⅰ22事件）は、本記述と類似の事案において、「被害者が身体傷害を承諾したばあいに傷害罪が成立するか否かは、単に承諾が存在するという事実だけでなく、右承諾を得た動機、目的、身体傷害の手段、方法、損傷の部位、程度など諸般の事情を照らし合せて決すべきものであるが、本件のように、過失による自動車衝突事故であるかのように装い保険金を騙取する目的をもって、被害者の承諾を得てその者に故意に自己の運転する自動車を衝突させて傷害を負わせたばあいには、右承諾は、保険金を騙取するという違法な目的に利用するために得られた違法なものであって、これによって当該傷害行為の違法性を阻却するものではない」と判示した。承諾が傷害罪の違法性を阻却しない以上、甲が乙の承諾を誤信していたとしても、甲に傷害罪（204条）が成立する。

📚 根拠条文・判例

→ 刑法204条（傷害罪） → 最決昭55.11.13（違法目的の傷害承諾＝無効） → 被害者の承諾の体系（承諾無効型）

👨‍🏫 教授の解説

1ポイント

要するに、違法な目的に利用するために得た傷害の承諾は有効かを問うています。最決昭55.11.13は、保険金詐取目的で得た承諾は違法性を阻却しないと判示。承諾が無効である以上、それを誤信したことも結論を左右しません。

2考え方の道筋

承諾の有効性判断：傷害罪における承諾の有効性は、承諾の存在だけでなく動機・目的・手段・方法・傷害の部位程度を総合して判断する。保険金詐取という違法な目的に利用するための承諾は、社会的相当性を欠き違法性を阻却しない＝無効である。

承諾の誤信の扱い：そもそも傷害罪の違法性を阻却し得ない承諾だから、甲がその承諾を誤信していたか否かは犯罪成否を左右しない。承諾が無効な罪（エ）と承諾が無意味な罪（ア）は、いずれも「承諾の誤信があっても成立する」という同じ帰結に至る。よって傷害罪が成立する。

✓ 正（交通事件原票の供述書は名義人の承諾の有無で偽造該当性が左右されず私文書偽造罪が成立）
最決昭56.4.8は、交通事件原票中の供述書は性質上作成名義人以外が作成できず、他人名義で作成すればあらかじめ承諾を得ていても私文書偽造罪が成立するとする。名義人の承諾の有無は「偽造」該当性を左右しないから、甲が承諾を誤信しても偽造の認識に欠けず私文書偽造罪が成立する。
  1. 偽造＝名義人と作成者の人格の同一性の偽り：私文書偽造罪の「偽造」は名義人と作成者の人格の同一性を偽る点にある。本来は名義人の承諾があれば原則として偽造に当たらない。
  2. 交通事件原票供述書の特殊性：最決昭56.4.8は、交通事件原票中の供述書は文書の性質上作成名義人以外の者が作成することが法令上許されず、他人名義での作成は承諾があっても偽造に当たるとする。
  3. 承諾が要件を左右しない型：承諾の有無が「偽造」該当性を左右しないから、承諾の誤信は偽造という構成要件該当事実の認識を妨げない。故意は否定されず成立する。
記述原文

オ　甲は、交通違反の取締りを受けた際に乙の氏名を名乗ることについての乙の承諾がないのに、これがあると誤信して、交通違反を警察官に現認された際、乙の氏名を名乗り、交通反則切符の供述書に乙の名義で署名押印した。（有印私文書偽造罪）

📖 解説原文

一般的に、名義人から名義の使用について承諾がある場合には、私文書偽造罪（159条）は成立しない。しかし、最決昭56.4.8（百選Ⅱ97事件）は、「交通事件原票中の供述書は、その文書の性質上、作成名義人以外の者がこれを作成することは法令上許されないものであって、右供述書を他人の名義で作成した場合は、あらかじめその他人の承諾を得ていたとしても、私文書偽造罪が成立する」と判示した。これは、交通事件原票中の供述書について、名義人の承諾の有無により私文書偽造罪の構成要件である「偽造」に当たるか否かは左右されないという趣旨である。よって、甲が名義人乙の承諾を得たと誤信しても、偽造という構成要件該当事実の認識に欠けるところはないから、故意は否定されず、甲に有印私文書偽造罪が成立する。

👨‍🏫 教授の解説

1ポイント

要するに、名義人の承諾があっても「偽造」に当たる文書かを問うています。最決昭56.4.8は、交通事件原票の供述書は性質上名義人本人しか作成できず、承諾があっても他人名義作成は偽造とする。承諾が要件を左右しない以上、承諾の誤信も偽造の認識を妨げません。

2考え方の道筋

原則と例外：私文書偽造罪の「偽造」は名義人と作成者の人格の同一性を偽ること。通常は名義人の承諾があれば人格の同一性の偽りがなく偽造に当たらない。ところが交通事件原票の供述書は、性質上作成名義人以外の作成が法令上許されない文書であり、承諾があっても他人名義での作成は偽造に当たる。

承諾の誤信の効果：承諾の有無が「偽造」該当性を左右しないから、甲が承諾を得たと誤信しても、他人名義で交通事件原票の供述書を作成しているという偽造の事実の認識に欠けない。故意は否定されず、有印私文書偽造罪が成立する。承諾が「構成要件該当性を左右しない」点で、住居侵入（ウ）と対照的な処理になる。

🧭 被害者の承諾の体系（本問の俯瞰）★★★

判定の二段階①その罪・その被害者で承諾が法的に意味を持つかを判定し、②意味を持つ場合に承諾の有効性・承諾に関する錯誤の効果を検討する。承諾の体系的位置（構成要件阻却か違法性阻却か）で錯誤の効果が変わる。
