刑法38条3項は、法律を知らなかったことで故意（罪を犯す意思）が阻却されることはないとした上で、「情状により、その刑を減軽することができる」と規定するのみ。刑の免除はできない。記述アは「減軽し、又は免除することができる」とする点で条文に反し誤り。
  1. 違法性の意識と故意：法の不知でも故意（罪を犯す意思）は阻却されない（38条3項本文・違法性の意識不要説＝判例）。
  2. ただし書の効果の限界：38条3項ただし書の効果は「情状により減軽することができる」のみで、「免除」は含まれない（任意的減軽どまり）。
  3. ひっかけの型：条文にない「又は免除」を付加して効果を膨らませる罠。「減軽」と「免除」の効果区別を正確に押さえることが核心。

📖 解説原文

記述アは誤り（正解は2）。前段の「法律を知らなかったとしても、そのことによって、罪を犯す意思がなかったとすることはできない」は刑法38条3項本文どおりで正しい。すなわち違法性の意識（の可能性）は故意の要素ではないとするのが判例（大判昭和9年・最判等の確立した立場）であり、法の不知は故意の存否と無関係である。問題は後段の効果の部分である。同条同項ただし書は「情状により、その刑を減軽することができる」と規定するのみで、刑の免除を認めていない。記述アは「情状により、その刑を減軽し、又は免除することができる」とする点で、条文にない「免除」を付加しており誤りである。法の不知は故意の存否には影響しないが、責任非難の程度（責任判断）には影響し得るため任意的減軽の余地を残した、というのが38条3項ただし書の趣旨である（減軽にとどまり免除には及ばない）。

📚 根拠条文・判例

→ 刑法38条3項（法律の不知・違法性の意識） → 刑法39条（責任能力・対比）

核心は効果の限界＝38条3項ただし書は「減軽」のみで「免除」を含まない。違法性の意識不要説（判例）の下でも、法の不知が責任非難を減じ得る場合に任意的減軽の余地を認めるにとどまる。記述アの「又は免除」が条文逸脱。

👨‍🏫 教授の解説

1ポイント

要するに、法の不知（違法性の意識を欠くこと）が故意・責任に与える効果は何か、そして38条3項ただし書の効果は「減軽」止まりか「免除」まで及ぶかを聞いています。記述アは前段（故意は阻却されない）は正しいが、後段で条文にない「又は免除」を付加した点が38条3項に反する致命的誤り。条文の効果文言を正確に記憶しているかを問う典型問題。

2考え方の道筋

違法性の意識と故意の関係：判例は違法性の意識不要説（厳格な故意説を採らない）に立ち、行為者が自己の行為を法律上許されないと知らなくても（法の不知）、構成要件該当事実の認識がある限り故意は成立するとする。38条3項本文「法律を知らなかったとしても、そのことによって、罪を犯す意思がなかったとすることはできない」はこの立場を条文化したものである。したがって記述ア前段は正しい。

ただし書の効果範囲：38条3項ただし書は「情状により、その刑を減軽することができる」とのみ規定する。これは、法の不知が故意を阻却しないとしても、違法性の意識を欠いたことが行為者の責任非難の程度を減じ得る場合に、裁判所の裁量で刑を軽くする余地を残したものである。しかし効果はあくまで任意的「減軽」に限られ、「免除」は含まれない。

記述アの誤りの所在：記述アは「情状により、その刑を減軽し、又は免除することができる」とする。前半の「減軽」は正しいが、後半の「又は免除」は条文に存在しない。減軽と免除は刑法上明確に区別される効果であり（例えば中止犯〔43条但書〕は「減軽し、又は免除する」必要的減免、過剰防衛〔36条2項〕は「減軽し、又は免除することができる」任意的減免）、38条3項は減軽のみを定める。したがって記述アは誤り。

👨‍🏫 教授の解説

1ポイント

要するに、心神喪失・心神耗弱の該当性は誰がどう判断するのか（鑑定医に従うのか、裁判所が決めるのか）を聞いています。判例（最決昭59.7.3）は、該当性は法律判断であって専ら裁判所の判断に委ねられるとし、鑑定書の内容に加えて犯行当時の病状・犯行前の生活状態・動機態様等を総合して認定してよいとした。記述イはこれにそのまま合致し正しい。

2考え方の道筋

責任能力の判断構造（混合的方法）：刑法39条の心神喪失とは、精神の障害により、行為の違法性を弁識し（弁識能力）、その弁識に従って行動を制御する能力（制御能力）を欠く状態をいい、心神耗弱とはこれらの能力が著しく減退した状態をいう。判断方法は、生物学的要素（精神の障害の有無・程度）と心理学的要素（弁識・制御能力の有無・程度）を組み合わせる混合的方法による。

鑑定と裁判所の関係：精神の障害という生物学的要素については精神医学の鑑定が重要な資料を提供する。しかし、その障害ゆえに弁識能力・制御能力が失われ又は著しく減退していたか（＝心神喪失・心神耗弱に当たるか）という最終評価は、規範的・法律的判断であり、裁判所の専権に属する。したがって裁判所は鑑定意見に法的に拘束されず、鑑定の結論と異なる責任能力判断をすることも許される（鑑定の証明力を裁判所が評価する）。

記述イの正しさ：記述イは「心神喪失又は心神耗弱に該当するかどうかは法律判断であって、専ら裁判所の判断に委ねられており、犯行当時の病状、犯行前の生活状態、犯行の動機・態様等を総合して判断される」とする。これは最決昭59.7.3の判示（責任能力該当性は法律判断・裁判所の専権・諸事情の総合判断）と逐語的に一致するため正しい。

旧40条の内容と趣旨：改正前刑法40条は「瘖唖者ノ行為ハ之ヲ罰セス又ハ其刑ヲ減軽ス」と規定し、瘖唖者の行為を必要的に不罰又は減軽するものであった。制定当時（明治40年）は、聴覚・言語障害者は適切な教育を受けられず是非弁別能力が一般に未発達であるとの想定に基づいていた。

削除の理由：その後、聾教育の普及・発達により、瘖唖者であっても弁識能力・制御能力が必ずしも減退しないことが明らかになった。瘖唖という身体的属性のみを理由に一律に減免することは、①能力の実質を問わずに画一的に扱う点で責任主義（責任能力は能力の実質で判断する）と整合せず、②障害者に対する不当な差別的取扱いであるとの強い批判を受けた。そこで平成7年の刑法一部改正（漢字カタカナ文語体を口語体に改める改正）の機会に、旧40条は削除された。

記述ウの誤り：記述ウは「先天的に耳が聞こえない者の行為については、必要的にその刑を減軽し、又は免除する」とするが、これは削除された旧40条の規律であり、現行法には存在しない。現在は瘖唖者であっても39条の心神喪失・心神耗弱に当たるか否かで個別に責任能力が判断され、これに当たらなければ完全責任能力が認められる。したがって記述ウは誤り。

📖 解説原文

記述エは誤り（正解は2）。刑法41条は「14歳に満たない者の行為は、罰しない」と規定する。これは責任年齢に関する規定であり、14歳未満の者（刑事未成年者）については、現実に行為の是非善悪を弁識する能力やその弁識に従って行動を制御する能力が備わっているか否かを一切問わず、一律に責任無能力者として扱い、刑罰を科さない。これは、心身の発達途上にある低年齢者には可塑性があり犯罪予防の見地から刑罰よりも保護・教育的処遇（少年法による保護処分等）が適当であるという刑事政策的判断に基づき、責任能力の有無を年齢という画一的・生物学的基準で割り切ったものである。したがって、14歳未満の者については、たとえ知能が発達し是非善悪の弁識能力やそれに従った行動制御能力が十分に認められる場合であっても、刑法上は絶対的に処罰されない。記述エは「そのような者については処罰されることがある」とする点で、41条が定める画一的不処罰の趣旨に反し誤りである。

📚 根拠条文・判例

→ 刑法41条（責任年齢・14歳未満不処罰） → 刑法39条（心神喪失・心神耗弱との対比）

核心は41条の画一性。責任年齢は能力の実質を問わず年齢のみで割り切る画一的基準。14歳未満なら弁識能力が十分でも一律不処罰（絶対的責任無能力）。記述エの「処罰されることがある」は41条に正面から反する。

1ポイント

  • 是非善悪の弁識能力等が現実に十分でも、14歳未満なら一律に処罰されない（絶対的責任無能力）

2考え方の道筋

責任能力否定の二経路：刑法は責任能力（弁識能力・制御能力）を欠く者を罰しないが、その否定には二つの経路がある。第一は39条の心神喪失・心神耗弱で、これは精神の障害と弁識・制御能力の有無を実質的に審査する（混合的方法）。第二は41条の責任年齢で、これは14歳未満という年齢のみで画一的に責任能力を否定する。

41条の画一性の趣旨：41条が能力の実質を問わず画一的に14歳未満を不処罰とするのは、①低年齢者は心身ともに発達途上で人格の可塑性が大きく、刑罰による非難よりも保護・教育による改善が犯罪予防上適切であること（刑事政策的考慮）、②個々の少年の能力を実質審査することは困難で基準が不安定になること、を考慮したものである。そこで法は「14歳」という明確な線引きを採用し、その線を下回る者は能力のいかんを問わず一律に刑罰を科さないこととした。

記述エの誤り：記述エは「14歳未満でも弁識能力等が十分なら処罰されることがある」とする。しかし41条は能力の実質を問わない画一的規定であり、14歳未満であればどれほど精神的に成熟していても刑罰は科されない（少年法上の保護処分等は別途あり得るが、これは「刑罰」ではない）。したがって記述エは誤り。なお、14歳以上であれば原則として責任能力が認められるが、その場合でも39条の心神喪失・耗弱に当たれば不罰・減軽される。

✓ 正しい（正解は 1）── 刑法42条2項の首服
刑法42条2項は、告訴がなければ公訴を提起できない罪（親告罪）について、告訴をすることができる者に対して自己の犯罪事実を告げその措置にゆだねたときも、42条1項（自首）と同様に刑を減軽することができると定める。これを首服という。記述オはこの首服の規定に合致し正しい。

  2. 効果は任意的減軽：「刑を減軽することができる」（免除は含まない）。自首（1項）と首服（2項）で効果は共通。
  3. 区別の核心：自首＝捜査機関へ申告／首服＝親告罪で告訴権者へ申告。申告先の違いで概念が分かれる点を押さえる。
記述オ原文

親告罪について、告訴権者に対して自己の犯罪事実を告げ、その措置に委ねたときは、刑を減軽することができる。

📖 解説原文

記述オは正しい（正解は1）。刑法42条は自首・首服による刑の任意的減軽を定める。同条1項は、罪を犯した者が捜査機関に発覚する前に自首したときは、その刑を減軽することができると規定する（自首）。これに対し同条2項は、「告訴がなければ公訴を提起することができない罪」すなわち親告罪について、「告訴をすることができる者に対して自己の犯罪事実を告げ、その措置にゆだねたとき」も1項と同様とすると規定する。これを首服（しゅふく）という。親告罪では告訴権者の意思が訴追の前提となるため、犯人が捜査機関ではなく告訴権者に対して自らの犯罪事実を申告し、訴追するか否かの判断をその者にゆだねることに、自首に準じた刑の減軽の利益を認めたものである。記述オは「親告罪について、告訴権者に対して自己の犯罪事実を告げ、その措置に委ねたときは、刑を減軽することができる」とするもので、42条2項の首服の内容と合致するため正しい。効果は自首と同じく任意的減軽（「することができる」）であり、必要的減軽でも免除でもない点も条文どおりである。

📚 根拠条文・判例

→ 刑法42条（自首・首服） → 刑法38条3項（効果＝任意的減軽の対比）

核心は首服＝42条2項。親告罪につき告訴権者に犯罪事実を告げ措置にゆだねたとき、自首（1項）と同様に刑を任意的に減軽できる。記述オは条文どおりで正しい。効果は「減軽することができる」（任意的減軽・免除なし）。

👨‍🏫 教授の解説

1ポイント
  • 刑法42条2項は首服（親告罪で告訴権者に犯罪事実を告げその措置にゆだねること）を定める
  • 効果は自首（1項）と同様に刑を減軽することができる（任意的減軽）
  • 記述オはこの首服の規定内容そのままで正しい

本記述は42条2項（首服）の知識を問う。自首（1項・捜査機関への申告）と首服（2項・親告罪で告訴権者への申告）の区別、および効果が「任意的減軽」である点を押さえていれば正しいと判断できる。

2考え方の道筋

自首（42条1項）：罪を犯した者が、捜査機関に発覚する前に自ら捜査機関に対して犯罪事実を申告し、その訴追を含む処分を求めることをいう。これにより捜査の端緒が得られ犯人の改悛も窺われることから、刑を減軽することができる（任意的減軽）。

首服（42条2項）：親告罪（告訴がなければ公訴を提起できない罪）については、訴追の前提として告訴権者の告訴が必要である。そこで、犯人が捜査機関にではなく告訴をすることができる者（告訴権者）に対して自己の犯罪事実を告げ、訴追するか否かの措置をその者にゆだねたときも、自首に準じて刑の減軽を認めるのが相当である。これが首服であり、効果は1項の自首と同様に任意的減軽である。親告罪では告訴権者の意思が訴追を左右するため、その者への申告に自首類似の意義を認めたものである。

記述オの正しさ：記述オは「親告罪について、告訴権者に対して自己の犯罪事実を告げ、その措置に委ねたときは、刑を減軽することができる」とする。これは42条2項の首服の要件（①親告罪であること②告訴権者に自己の犯罪事実を告げること③その措置にゆだねること）と効果（任意的減軽）を正確に述べており、条文どおりで正しい。本問でイと並ぶ「正しい（1）」の記述である。
