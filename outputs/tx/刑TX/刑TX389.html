  2. 密室・少人数でも公然：不特定多数を勧誘して集めた以上、夜間一室を密閉してなされても公然性を妨げない（最決昭31.3.6）。

路上で不特定多数を勧誘して客を集め、ビル1室で出入りを制限した状態で全裸性行為を6名に有料観覧させた場合、甲に公然わいせつ罪の共同正犯は成立しない。

📖 解説原文

本記述は誤り。公然わいせつ罪（刑法174条）にいう「公然」とは、不特定又は多数の人が認識することのできる状態をいう（最決昭32.5.22）。そして判例は、不特定多数の人を勧誘した結果集まったそれぞれ数十名の客の面前でわいせつ行為をした事案に関し、当該わいせつ行為が「たとえ夜間一定の部屋を密閉してなされたとしても公然わいせつ罪の成立を妨げるものではない」としている（最決昭31.3.6）。本記述では、甲は不特定多数の通行人を勧誘して客を集めている以上、全裸で性行為を行うというわいせつ行為を、6名の客のみに外部との出入りが制限された部屋で観覧させたにすぎないとしても、観覧者は「不特定」の客であって、公然性は認められる。したがって甲（および雇用された男女）には公然わいせつ罪の共同正犯（174条・60条）が成立する。「共同正犯は成立しない」とする本記述は判例の立場に反し、誤りである。

📚 根拠条文・判例

→ 刑法174条（公然わいせつ罪） → 最決昭31.3.6・最決昭32.5.22（公然性の意義）

「公然」性は観覧者の人数の多寡ではなく「不特定」性で判定される。客が6名でも、不特定の客を勧誘して集めた以上、密閉空間でも公然性は失われない。共同正犯成立を否定する本記述は誤り。

👨‍🏫 教授の解説

1ポイント

要するに、「公然」の判定が観覧者の人数（6名）や場所の閉鎖性（出入り制限・密室）で否定されるのかを聞いています。判例は公然性を「不特定又は多数の人が認識できる状態」と定義し、不特定性が認められれば少人数・密室でも公然性は失われないとする。誘客行為に着目すれば本問の客は「不特定」であり、公然性が肯定される。

2考え方の道筋

「公然」の定義の精密把握：最決昭32.5.22は公然を「不特定又は多数の人が認識することのできる状態」と定義する。「不特定」と「多数」は選択的要件であり、いずれか一方が満たされれば公然性は肯定される。したがって観覧者が6名と少数でも、その客が「不特定」である限り公然性は認められる。

密室・出入り制限の評価：最決昭31.3.6は、不特定多数を勧誘して集めた客の面前でのわいせつ行為につき「たとえ夜間一定の部屋を密閉してなされたとしても公然わいせつ罪の成立を妨げない」とした。場所を密閉し外部との出入りを制限しても、誘客の段階で不特定性が確定している以上、公然性は維持される。閉鎖空間化は公然性を覆す事情にならない。

共同正犯の成否：甲は自らが雇用した男女に全裸で性行為を行わせ、それを有料観覧させて利益を得ている。甲は実行行為者（男女）と共同して公然わいせつ行為を実現しており、共謀に基づく一部実行・全部責任（60条）として公然わいせつ罪の共同正犯が成立する。本記述の「共同正犯は成立しない」は結論を誤っている。

  2. 受け手主導型でも頒布：顧客のDL操作を契機とするものでも、記録・保存に至らしめれば頒布に当たる。

記述原文

甲がわいせつ画像データをサーバに送信・蔵置させ、不特定多数の顧客がDLして自己のPCに記録・保存することを可能にし、現に顧客がDLして保存した場合、甲にわいせつ電磁的記録等送信頒布罪が成立する。

📖 解説原文

本記述は正しい。わいせつ電磁的記録等送信頒布罪（175条1項後段）における「頒布」とは、「不特定又は多数の者の記録媒体上に電磁的記録その他の記録を存在するに至らしめること」をいう（最決平26.11.25・百選Ⅱ101事件）。そして同判例は、「不特定の者である顧客によるダウンロード操作を契機とするものであっても、その操作に応じて……わいせつな動画等のデータファイルを当該顧客のパーソナルコンピュータ等の記録媒体上に記録、保存させることは、刑法175条1項後段にいうわいせつな電磁的記録の『頒布』に当たる」とする。本記述では、甲はわいせつ画像データをサーバに送信して記憶・蔵置させた上、不特定多数の者がインターネットを経由してこれをダウンロードしてパソコンに記憶・保存することを可能にし、現に顧客がわいせつ画像データをダウンロードしてパソコンに記憶・保存しているから、わいせつ電磁的記録等送信頒布罪が成立する。記録・保存する者（顧客）においてダウンロード等の作業をする必要があったとしても、結論は左右されない（判例も同旨）。よって「成立する」とする本記述は正しい。

📚 根拠条文・判例

→ 刑法175条1項後段（電磁的記録送信頒布罪） → 最決平26.11.25（百選Ⅱ101・頒布の意義）

電磁的記録の「頒布」＝不特定又は多数の者の記録媒体上に存在させること。顧客側のDL操作を契機とする「受け手主導型」でも、その操作に応じて記録・保存させれば頒布に当たる（平23改正で新設された後段の射程）。

👨‍🏫 教授の解説

1ポイント

要するに、サーバに置いた画像を顧客が自分でダウンロードする「受け手主導型」配信が175条1項後段の『頒布』に当たるかを聞いています。最決平26.11.25は、頒布を「不特定又は多数の者の記録媒体上に電磁的記録を存在するに至らしめること」と定義し、顧客のDL操作を契機とする場合でも頒布に当たると明言した。本記述はこの判例と整合し、正しい。

2考え方の道筋

平成23年改正と「頒布」概念の拡張：かつて旧175条は「頒布・販売・公然陳列」を対象とし、サーバ蔵置型の配信は「陳列」か「頒布」かで争いがあった。平成23年改正で175条1項後段（電気通信の送信による電磁的記録の頒布）が新設され、ネット配信が明文で捕捉された。最決平26.11.25は新設後段の「頒布」の意義を「不特定又は多数の者の記録媒体上に存在するに至らしめること」と画定した。

「受け手主導型」でも頒布に当たる：問題は、甲が能動的に送りつけるのではなく、顧客が自らDL操作をして取得する形態でも「頒布した」と言えるかである。判例は、顧客のDL操作を契機とするものであっても、その操作に応じてデータを顧客のPCの記録媒体上に記録・保存させれば頒布に当たるとした。送信の物理的トリガーが受け手側にあっても、甲がサーバに蔵置して不特定多数が取得可能な状態を作り、現に記録・保存に至らしめている以上、頒布の実質を備える。

本記述の但書部分の評価：本記述は「記録・保存する者において、記録・保存の際、画像データのダウンロード等の作業をする必要があったとしても」成立するとする。これは判例が明示的に否定した反対論（受け手の作業が介在すれば頒布でないとする立場）を排斥する趣旨であり、判例の結論と一致する。よって本記述は正しい。

📚 根拠条文・判例

→ 刑法175条2項（有償頒布目的所持罪） → 富山地判平2.4.13（マスター所持も有償頒布目的所持）

「有償で頒布する目的」は所持物それ自体の頒布目的に限定されない。複写物を販売する目的でマスター（原盤）を所持していれば、原盤は複写・頒布の手段として所持されており有償頒布目的がある。

👨‍🏫 教授の解説

1ポイント

要するに、「有償で頒布する目的」とは、所持しているその物（マスターDVD）自体を売る目的に限られるのかを聞いています。裁判例（富山地判平2.4.13）は、ダビングのみを販売する意思でマスターを所持した場合も有償頒布目的所持罪を構成するとし、複写物を販売する目的でも「有償で頒布する目的」があるとした。マスター自体を売らなくても所持罪は成立する。

2考え方の道筋

「有償で頒布する目的」の解釈：175条2項の有償頒布目的所持罪は、わいせつ物の市場への供給の前段階（所持・保管）を捕捉する目的犯である。保護法益（健全な性風俗の維持）からすれば、頒布される現物が所持物そのものか複写物かで処罰の要否が変わる理由はない。所持物を素材として複写物を量産し市場に供給する目的こそ、規制が最も妥当する典型である。

原盤（マスター）の役割：マスターDVDは、それ自体を市場に流通させなくても、複写によって無数の頒布用DVDを生み出す原資として所持されている。甲は「客の注文に応じて他のDVDに複写して販売する」意図を有しており、マスターはまさに有償頒布の手段として所持されている。富山地判平2.4.13がダビング販売目的のマスター所持を有償頒布目的所持罪としたのはこの趣旨である。

本記述の誤り：本記述は「マスターDVD自体を販売する意図はなかった」ことを理由に所持罪の不成立を導くが、これは「有償で頒布する目的」を所持物それ自体の頒布目的に限定する誤った前提に立っている。複写販売目的でも有償頒布目的があるから、所持罪は成立する。よって本記述は誤り。

「頒布」の既遂時期：175条1項前段の「頒布」とは、不特定又は多数の者に対しわいせつ物を現実に交付することをいう。交付の約束（売買契約の成立）や代金の受領、発送手続の完了は、頒布に向けた準備・着手にとどまり、相手方の支配領域にわいせつ物が現実に到達して初めて頒布が完成する。判例（大判昭11.1.31・最判昭34.3.5）が「交付の約束では足りず現実の交付を要する」とするのはこの趣旨である。
