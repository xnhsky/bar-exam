❀A-1 問題文

文書等毀棄罪に関する次の1から5までの各記述を判例の立場に従って検討した場合、正しいものはどれか。

1公用文書等毀棄罪における「公務所の用に供する文書」とは、公務所又は公務員が作成したもので、現に公務所において使用され、又は使用の目的をもって保管されている文書のことをいう。
2偽造された文書や未完成の文書は、公用文書等毀棄罪の客体とはなり得ない。
3保存期間が経過した後の文書は、公用文書等毀棄罪の客体とはなり得ない。
4私用文書等毀棄罪における「権利又は義務に関する他人の文書」とは、権利又は義務の存否・得喪・変更・消滅等を証明し得る他人所有の文書のことをいう。
5手形や小切手等の有価証券は、私用文書等毀棄罪の客体とはなり得ない。

（参照条文）刑法

第258条（公用文書等毀棄）　公務所の用に供する文書又は電磁的記録を毀棄した者は、3月以上7年以下の拘禁刑に処する。

第259条（私用文書等毀棄）　権利又は義務に関する他人の文書又は電磁的記録を毀棄した者は、5年以下の拘禁刑に処する。

第262条（自己の物の損壊等）　自己の物であっても、差押えを受け、物権を負担し、賃貸し、又は配偶者居住権が設定されたものを損壊し、又は傷害したときは、前三条の例による。

✗ 誤（「作成者」を限定する点が判例に反する）
最判昭38.12.24は、258条の「公務所の用に供する文書」を作成者・作成目的を問わず、現に公務所で使用に供せられ又は使用目的で保管される文書の総称とする。本記述は「公務所又は公務員が作成したもの」と作成主体を限定する点で判例に反し誤り。客体の関所を不当に狭めている。
  1. 判例の定義：最判昭38.12.24は258条の客体を「作成者・作成目的を問わず、現に公務所で使用・保管される文書」とする。
  2. 本記述の誤り：「公務所又は公務員が作成したもの」と作成主体を限定する点が判例に反する。私人作成・公務所保管型を漏らす。
  3. 視点：155条（偽造罪）の作成名義基準を258条（毀棄罪）に流用してはならない。毀棄罪は現用状態のみを見る。
記述原文

公用文書等毀棄罪における「公務所の用に供する文書」とは、公務所又は公務員が作成したもので、現に公務所において使用され、又は使用の目的をもって保管されている文書のことをいう。

📖 解説原文

最判昭38.12.24は、「刑法258条にいわゆる『公務所ノ用ニ供スル文書』とは、その作成者、作成の目的等にかかわりなく、現に公務所において使用に供せられ、又は使用の目的をもつて保管されている文書を総称するものと解すべきである」とする。すなわち判例は、文書を誰が作成したか・何のために作成したかを客体性の要件から外し、現に公務所が使用・保管しているという現在の状態のみを基準とする。これは、本罪の客体が「公務所の用に供する文書」であって、公文書偽造罪（155条）にいう「公務所が作成名義人となるべき文書」とは区別されることに由来する。本記述は「公務所又は公務員が作成したもの」という作成主体の限定を付加する点で、判例の総称的定義より客体を狭く画しており誤り。私人が作成した申請書・届出書であっても公務所が受理・保管していれば客体となる点が、この限定では捉えられない。

📚 根拠条文・判例

→ 刑法258条（公用文書等毀棄罪） → 最判昭38.12.24（公務所の用に供する文書の定義） → 刑法155条との区別（作成名義人）

判例の定義＝作成者・作成目的を問わない『現用・保管』基準。本記述の「公務所又は公務員が作成したもの」は客体を作成主体で限定する誤った付加で、私人作成文書の受理・保管型を漏らす。

👨‍🏫 教授の解説

1ポイント

要するに、258条の客体である「公務所の用に供する文書」に作成主体（公務所・公務員が作った文書）という縛りがあるかを聞いています。最判昭38.12.24は作成者・作成目的を一切問わず、「現に公務所が使用・保管していること」だけを基準とするので、本記述の作成主体限定は判例と真っ向から食い違う。

2考え方の道筋

保護法益からの逆算：258条は公務所における文書の現実の使用・利用可能性を保護する。誰が作ったかではなく、いま公務の用に供されているかが決定的である。だからこそ判例は「作成者、作成の目的等にかかわりなく」と明言する。

155条（公文書偽造罪）との対比：偽造罪は「公務所・公務員が作成名義人となるべき文書」を保護し、ここでは名義（作成主体）が決定的に重要となる。毀棄罪と偽造罪は保護の着眼点が異なり、毀棄罪では作成名義は問題とならない。本記述はこの偽造罪の発想を毀棄罪に持ち込む誤りである。

具体例での検証：私人が役所に提出した申請書・確定申告書は、私人が作成したものであるが、公務所が受理して使用・保管している以上、判例の定義では258条の客体となる。本記述の「公務所又は公務員が作成したもの」という限定では、これらの私人作成・公務所保管型の文書が客体から漏れてしまう。

→ 大判大9.12.17（偽造徴税伝令書） → 最判昭52.7.14（未完成文書） → 刑法258条（公用文書等毀棄罪）

👨‍🏫 教授の解説

1ポイント

要するに、「ニセモノだから」「未完成だから」という理由で258条の客体から外れるかを聞いています。判例は、偽造文書（大判大9.12.17）も未完成文書（最判昭52.7.14）も客体となるとしており、本記述の一律排除は二重に誤り。

2考え方の道筋

偽造文書の処理：偽造文書は名義の真正を欠くが、258条が守るのは名義の真正ではなく現実の使用・保管機能である。徴税伝令書が偽造であっても、徴税のために役場で使われ保管されていれば、公務の遂行に現に役立っており、これを毀棄すれば公務が阻害される。だから偽造であることは客体性を否定しない。

未完成文書の処理：作成中の文書も、「文書としての意味、内容を備えるに至つた」段階に達すれば、公務所が現に使用している文書として客体となる（最判昭52.7.14）。完成・署名押印の有無ではなく、意味内容の具備が基準である。

共通の発想：肢1と同じく、258条の客体判定は「作成の局面（誰が・どう作ったか・完成したか）」ではなく「現用の局面（いま公務に供され得るか）」を見る。偽造・未完成という作成局面の瑕疵は、現用機能を否定しない限り客体性に影響しない。

2考え方の道筋

保存期間と保管状態の切り分け：保存期間は、公務所が文書を保管すべき義務期間を定める行政的な取扱いである。期間が満了しても、現実に文書が公務所内に存置されている限り、それは「保管されている」状態が継続している。大判明42.7.8はこの事実状態と法令上の取扱いの区別を前提に、廃棄前であれば客体となるとした。

肢1・2との共通発想：258条の客体判定は一貫して「現に公務所の用に供されているか・保管されているか」という現実の機能・状態を基準とする。肢1（作成者）、肢2（真正・完成度）に続き、本肢は「保存期間」という時間的要素で客体を狭めようとするが、いずれも判例の現用・保管基準より客体を狭める誤りである。

客体喪失の時点：文書が258条の客体でなくなるのは、現実に廃棄され公務所の管理を離れた時点である。保存期間の満了はその前段階にすぎず、満了と廃棄を同視してはならない。本記述は満了＝廃棄と誤って同視する。

記述原文

私用文書等毀棄罪における「権利又は義務に関する他人の文書」とは、権利又は義務の存否・得喪・変更・消滅等を証明し得る他人所有の文書のことをいう。

📖 解説原文

259条の「権利又は義務に関する他人の文書」とは、権利又は義務の存否・得喪・変更を証明するための文書をいい、本記述はこの定義を正確に述べるもので正しい。重要なのは、私文書偽造罪（159条）と異なり、単なる「事実証明に関する文書」を含まない点である。同規定（259条）の趣旨は、権利又は義務に関する文書等の財産的価値・効用を保護し、その他の文書よりも厚く保護しようとする点にある。159条が「権利、義務又は事実証明に関する文書」を広く客体とするのに対し、毀棄罪である259条は財産罪としての性格から客体を「権利又は義務に関する文書」に限定し、事実証明文書を除外する。また「他人の文書」とは、他人の所有に属するものをいう。もっとも、自己の所有物であっても、差押えを受け、物権を負担し、賃貸し、又は配偶者居住権が設定されたものは、262条により本罪の客体となる。本記述は259条の客体の定義として判例・通説の理解に合致し、本問において唯一正しい記述である。

📚 根拠条文・判例

→ 刑法259条（私用文書等毀棄罪） → 刑法262条（自己の物の特則） → 最決昭44.5.1（259条文書の範囲）

259条の客体＝権利義務の存否・得喪・変更等を証明し得る他人所有の文書。財産罪としての性格から事実証明文書を除外する点が偽造罪(159条)との決定的差異。

  • 私文書偽造罪（159条）と異なり、単なる「事実証明に関する文書」は含まない
  • 趣旨は権利義務文書の財産的価値・効用を厚く保護する点にある

本記述は259条の客体定義をそのまま正確に述べる。他の肢が客体を不当に狭めたり判例に反する中で、過不足のない定義を述べる本肢が唯一の正解である。

2考え方の道筋

259条と159条の客体範囲の対比：私文書偽造罪（159条）は「権利、義務又は事実証明に関する文書」を客体とし範囲が広い。これに対し私用文書等毀棄罪（259条）は「権利又は義務に関する他人の文書」に限られ、事実証明文書を含まない。毀棄罪が財産罪であり、保護すべきは文書の財産的価値・効用だからである。本記述は「権利又は義務の存否・得喪・変更・消滅等を証明し得る」と述べ、259条の限定を正確に捉えている。

「他人の文書」の意義：「他人の文書」とは他人の所有に属するものをいう。自己所有物は原則として客体とならないが、262条により、差押えを受け、物権を負担し、賃貸し、又は配偶者居住権が設定された自己所有物は本罪の客体となる。本記述は「他人所有の文書」と述べ原則を正確に示しており、262条の特則と矛盾しない。

判例による客体の広がり：259条の文書には、最決昭44.5.1が示すとおり有価証券（小切手）も含まれる（肢5参照）。これは「権利又は義務に関する」性質を有する限り、文書の形式を問わず広く客体に含めるものであり、本記述の定義と整合する。

✗ 誤（有価証券＝小切手も259条の文書に含まれる）
最決昭44.5.1は、259条の「権利、義務ニ関スル他人ノ文書」に有価証券である小切手も含まれるとする。有価証券は文書性を有する。本記述は有価証券を一律に客体から除外する点で判例に反し誤り。
  1. 判例（最決昭44.5.1）は259条の文書に有価証券である小切手も含まれるとする。
  2. 理由は有価証券が文書性を有すること（権利義務を表象する書面）。財産的効用は高い。
  3. 有価証券を一律に客体から除外する限定は、有価証券の文書性を肯定する判例の射程に反する。259条の客体定義（権利義務を証明し得る他人所有の文書）と表裏で理解できる。
記述原文

手形や小切手等の有価証券は、私用文書等毀棄罪の客体とはなり得ない。

📖 解説原文

本記述は手形・小切手等の有価証券を259条の客体から一律に除外するが、判例はこれを否定しており誤り。最決昭44.5.1は、「刑法259条の『権利、義務ニ関スル他人ノ文書』には、有価証券である小切手も含まれる」とする。判例の結論に賛成する学説は、その理由として、有価証券は文書性を有することを挙げる。有価証券は、財産上の権利が証券に化体され、権利の発生・移転・行使に証券を要するものであるが、その実体は権利義務関係を表示する書面であり、259条にいう「権利又は義務に関する文書」に該当する。手形・小切手は典型的に権利義務（金銭支払請求権）を証明・表象する文書であり、これを毀棄すれば権利者の財産的効用が害される。259条が権利義務に関する文書の財産的価値・効用を保護する趣旨に照らせば、有価証券を客体から除外する理由はない。本記述は有価証券を「文書ではない」と扱う点で、その文書性を肯定する判例と整合せず誤りである。

📚 根拠条文・判例

→ 最決昭44.5.1（小切手の文書性） → 刑法259条（私用文書等毀棄罪） → 刑法262条（自己の物の特則）

有価証券（小切手）は文書性を有し259条の客体に含まれる（最決昭44.5.1）。権利義務を表象する書面である以上、財産的効用を保護する259条の射程に入る。

👨‍🏫 教授の解説

1ポイント
  • 本記述は有価証券（手形・小切手）を259条の客体から一律に除外する
  • 最決昭44.5.1は、259条の文書に有価証券である小切手も含まれるとする
  • 理由＝有価証券は文書性を有する（権利義務を表象する書面）

有価証券というと「証券＝文書とは別物」と思いがちだが、判例は有価証券も文書の一種とみる。権利義務を表象する書面である以上、259条の客体に含まれる。

2考え方の道筋

有価証券の本質：有価証券は財産的権利が証券に化体されたものであるが、その物理的実体は権利義務関係を表示する書面（文書）である。小切手は支払委託・金銭債権を表象する文書であり、259条にいう「権利又は義務に関する他人の文書」の典型に当たる。

259条の趣旨からの帰結：259条は権利義務に関する文書の財産的価値・効用を保護する。手形・小切手はまさに財産的価値・効用の高い文書であり、これを毀棄すれば権利者の権利行使が妨げられる。保護の必要性は通常の権利義務文書より高いほどであり、客体から除外する理由はない。最決昭44.5.1はこの帰結を明示する。

肢4との表裏：肢4は259条の客体を「権利義務を証明し得る他人所有の文書」と正確に定義した。有価証券はこの定義に当然包含される。本肢5は同じ259条の客体から有価証券を誤って除外する点で、肢4の定義と正面から矛盾する。
