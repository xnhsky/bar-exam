# 刑法重問 Whisper文字起こし 校正用 共通語彙集（ASR誤変換マップ）

本ファイルは、アガルート重問・刑法講座の音声をWhisperで文字起こしした74ファイルの
誤変換訂正に使う**共通参照表**である。各処理エージェントは作業前に必ず本ファイルを読む。

## 大原則（厳守）

1. **音声認識エラー（同音・近音の誤変換、誤字、聞き取り崩れ）のみ**訂正する。
2. 講義の**内容・論理・文末表現・口調・フィラー（まあ／ね／んで／っていう 等）は一切変えない**。
   助詞・語尾・言いよどみ・繰り返しもそのまま残す。
3. **行構造を保持**する。原文は1セグメント＝1行。訂正は行内のトークン置換に限り、
   行の分割・結合・追加・削除をしない。**訂正後ファイルの行数は原文と完全一致**させること。
4. 講師の**言い間違い（口頭でのミス）は訂正対象外**。明らかなASRエラーのみ。
5. **法的な正誤判断に迷う箇所、文脈から確信が持てない箇所は、勝手に書き換えず**
   フラグメントの「要確認」に列挙する（修正前のまま残す）。
6. 元の74ファイルは**絶対に上書き・削除しない**。出力は `_corrected\` に同名で新規作成。

## B. 文脈依存（占いで一律置換せず、その箇所の意味で判断）

| 表記 | 判断 |
|---|---|
| 障害 / 傷害 | 身体への侵害＝**傷害**（傷害罪・傷害結果）。「障害物」「視覚障害」「故障」等は**障害**のまま。動物傷害も「傷害」 |
| 公 | **当事者を指すとき（公が／公は／公の罪責／公に）→「甲」**。「公務員・公務・公然・公共・公訴・公判・公道・公益」等は**公**のまま |
| 兵 | **当事者を指すとき→「丙」**。「兵庫・兵器・兵士」等は対象外 |
| 大津 | **当事者を指すとき→「乙」**（文脈で乙が登場する問題のみ） |
| 法制レベル / 法制 | 構成要件→違法性→責任 の体系上「違法性」を指すなら**違法性**、罪の成立要件を指すなら**構成要件**。文脈で判断 |
| 順序 | 条文準用の文脈→**準用**。手順・並び順の一般語なら**順序**のまま |
| 整備 | 「〜の整備（が問題）」で成立可否を指すなら**成否**。設備整備等は**整備**のまま |
| 意思 / 医師 / 意志 | 投げる「石（いし）」を指す問題では**石**。「意思（故意・防衛の意思）」は意思のまま。文脈必須 |
| 規範 / 既範 | 「規範」 |
| 既遂 / 帰趨 / 帰結 | 文脈で。結果発生＝既遂 |

## C. 問題ごとの固有名・固有事実（このファイル内の文脈からのみ判断）

各重問は別個の事例問題であり、登場人物・凶器・場所などは問題ごとに異なる。
- 当事者は原則 **甲・乙・丙・丁**（こう・おつ・へい・てい）。ASRが「公／兵／大津／低」等に
  崩すことがある。**その問題に実際に登場する当事者**に合わせて復元する。
- 固有の物・場所・行為の崩れは、**音が近く文脈上明確なときのみ**訂正し、必ずログに残す。
  少しでも不確実なら「要確認」へ。

## D. 出力仕様（各エージェント）

1. 各 `刑法_重問NN.txt` を Read し、上記方針で訂正。
2. 訂正版を **Write ツール**で `…\_corrected\刑法_重問NN.txt` に保存（UTF-8・原文と同一行数）。
3. 各ファイルの修正記録を、担当バッチの**フラグメント** `…\_corrected\_fragments\frag_BNN.md` に
   下記フォーマットで追記（1ファイル=1セクション）。

```
## 刑法_重問NN.txt
- 修正件数: X件 ／ 要確認: Y件 ／ 行数 原=___ 訂正後=___（一致）

### 修正一覧
| 行 | 修正前 | 修正後 | 理由 |
|---|---|---|---|
| 7 | 動物障害 | 動物傷害 | 身体への侵害は「傷害」。ASR同音誤変換 |
| 12 | 急迫性の侵害 | 急迫不正の侵害 | 正当防衛の要件語の聞き取り崩れ |
…

### 要確認
- 行NN「…」: 〇〇か△△か文脈から確定できず（理由）。原文のまま保持。
```

4. 行数が原文と一致しない場合は保存し直し、一致を確認してからフラグメントに記録する。
