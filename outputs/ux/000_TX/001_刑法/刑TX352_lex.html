【事例】── 放火罪をめぐる甲の罪責に関する4事例（○×評価）

ア甲は、木造アパートの空室の壁際に置いてあったダンボール箱に火をつけ、火を板壁に燃え移らせて放火したが、板壁の一部を焼損した時点で、アパートの住民に消し止められた。甲は、そのアパートに人が居住している部屋があることを認識していたが、人が居住する部屋に延焼するかもしれないとは認識しておらず、空室のみを焼損するつもりだった。甲に現住建造物等放火既遂罪が成立する。
イ甲は、家屋の居住者全員を殺害した後、証拠を隠滅するためにその家屋を焼失させようと考え、室内の布団に放火したが、布団を焼損した時点で、隣家の住民に消し止められた。甲に非現住建造物等放火未遂罪が成立する。
ウ甲は、妻と二人で自宅に居住していたが、甲の意図を知らない妻の旅行中、火災保険金を詐取する目的で自宅に放火して全焼させた。甲は、隣家に延焼することは予期していなかったが、隣家も延焼した。甲に延焼罪が成立する。
エ甲は、宿泊していたホテルの部屋に放火しようと考え、窓のカーテンに火をつけたが、カーテンを焼損した時点で、従業員に消し止められた。甲に現住建造物等放火既遂罪が成立する。

📖 この問題を物語で読む ── 放火罪は「何が燃えれば既遂か」「誰が住んでいれば現住か」で決まる

放火罪は、火という制御の難しい力を使って人の生命・身体・財産を一気に危険にさらす罪で、刑法はこれを重く扱います。中でも一番重いのが現住建造物等放火罪（108条）──人が住んでいる、または人がいる建物に火をつける類型です。これより軽いのが、人の住んでいない建物を焼く非現住建造物等放火罪（109条）。本問の四つの記述は別々の事案ですが、すべて同じ二本の物差し──「どこまで燃えたら既遂（焼損）になるか」と「その建物は現に人が住む建物か」──で測れます。この二本の物差しの当て方を一つずつ確かめていけば、各記述の正誤は自然に決まります。

まず一本目の物差し、焼損（既遂のライン）を押さえます。判例は、火が点火に使った媒介物（新聞紙やダンボールなど）を離れ、建物そのものが独立して燃え続ける状態に達したときに「焼損」があったとする独立燃焼説を採ります。建物の一部でも独立して燃え出せば、もう既遂です。逆に、まだ建物本体に燃え移っていなければ未遂にとどまります。ここで決定的に重要なのが、「燃えたその物が建物の一部かどうか」という線引きです。建物本体（板壁・柱など）が燃えれば焼損ですが、取り外しのきく置物が燃えただけでは、建物はまだ独立燃焼していないので既遂になりません。

この線引きを正面から扱うのが、二つの記述です。木造アパートの空室の板壁に火が燃え移り、その一部が焼けた事案では、板壁は建物そのものの一部ですから、独立燃焼に達した時点で焼損＝既遂です。さらに、外観上ひとかたまりの木造アパートは全体で一個の建造物（建造物の一体性）と見られ、空室部分に火をつけても、住人のいる部屋を含む建物全体への放火と評価されます。住んでいる部屋があると認識していた以上、空室だけ焼くつもりでも故意に欠けるところはなく、現住建造物等放火の既遂が成立します。これは正しい記述です。

対照的なのが、取り外せる物だけが燃えた二つの記述です。証拠隠滅のため室内の布団に火をつけて布団だけ焼けた事案では、布団は毀損せずに持ち出せる物で建物の一部ではないため、建物本体はまだ独立燃焼していません。よって既遂に達せず未遂にとどまります。ホテルの窓のカーテンに火をつけてカーテンだけ焼けた事案も同じで、容易に取り外せるカーテンは建物の一部とはいえず、その焼損だけでは建物が独立燃焼せず既遂にはなりません。放火罪でいう「建造物」の意味は、建造物損壊罪のそれと同じで、簡単に取り外せる物は含まないからです。カーテンの記述が「既遂が成立する」とした点は、ここで誤りになります。

次に二本目の物差し、現住性（人が住んでいるか）です。108条の「人」とは犯人以外の者を指し、犯人の家族も含みます。そして大切なのは、住人が一時的にいなくても、それだけで「人の住む建物」でなくなるわけではない、という点です。判例は現住性が失われるかどうかを、その建物の使用形態が変更されたかで判断します。日常の住居として使い続けられている限り、たまたま誰も在宅していない瞬間でも現住建造物のままです。

この物差しが効くのが、妻の旅行中に保険金目当てで自宅を全焼させた事案です。妻はあくまで旅行中で、帰宅すればまた住居として使うのですから、住居としての使用形態は変わっていません。よって自宅はなお現住建造物であり、甲には現住建造物等放火の既遂（108条）が成立します。ここで問題文は「延焼罪が成立する」としますが、これが誤りです。延焼罪（111条1項）は、もともと人の住まない建物や物への放火（109条2項・110条2項）から思いがけず燃え広がった場合の加重類型で、最も重い108条が成立する本件には出る幕がありません。一番重い罪が立つのに、軽い罪を前提とする延焼罪を持ち出すのは筋違いなのです。

こうして四つの記述は、二本の物差しできれいに腑分けできます。板壁の事案は「建物本体が独立燃焼した＝焼損」で既遂、布団とカーテンの事案は「燃えたのは取り外せる物で建物は燃えていない＝未遂」、保険金の事案は「一時不在でも現住性は残る＝最重の108条が立ち、延焼罪は不成立」。誤りはすべて、この線引きを取り違えたもの──建物の一部でない物の焼損を既遂と数えたり、現住建造物への放火に延焼罪をあてはめたりした点にあります。

本問の幹は二本──「燃えた物が建物の一部か」で焼損（既遂）の有無が決まる軸と、「一時不在でも使用形態が変わらなければ現住性は残る」という軸です。正しい記述とされる番号は、この二つの物差しを各事案に一つずつ当てた結果にすぎません。覚えるべきは答えの組合せではなく、独立燃焼説と取り外し可能物の線引き、そして現住性が使用形態で決まるという理由づけのほうです。

🎯 統合解説

📜 記述原文木造アパートの空室の壁際のダンボール箱に放火し、板壁の一部を焼損した時点で住民に消し止められた。甲は人が居住する部屋の存在は認識していたが空室のみを焼損するつもりだった。→ 甲に現住建造物等放火既遂罪が成立する。

💡 THE GIST○。外観上一個の木造アパートは建造物の一体性が肯定され、空室への放火も全体＝現住建造物への放火になる。板壁が独立燃焼に達した時点で焼損＝既遂で、現住部分の認識があれば故意も足りる。

1客体＝一体性放火対象部分が建造物全体と一体をなすかで客体が現住建造物（108条）か非現住かが決まる。外観上一個の木造アパートは一部放火でも全体への延焼蓋然性が高く一体性が肯定され、空室は現住建造物の一部となる（大判昭9.11.15）。

2既遂＝独立燃焼焼損は、火が媒介物を離れ目的物が独立して燃焼を継続する状態に達した時に認められる（独立燃焼説・最判昭23.11.2）。板壁の一部焼損は火がダンボールを離れ板壁が独立燃焼した状態であり既遂に達している。

3故意の射程放火の故意は客体が現住建造物（の一部）であることの認識で足り、現住区画への延焼の認識までは不要。居住部屋の存在を認識する甲には故意があり、「空室のみ焼損する意図」は故意を阻却しない。

💭 INTUITION一棟の家は「ひとつのかたまり」。隅の空室に火をつけても家全体に火を放ったのと同じ。板壁が自分の力で燃え始めた瞬間がゴールテープ＝既遂。

  1. 建造物の一体性：外観上一個の木造建物は一部放火でも全体＝現住建造物への放火と評価される（大判昭9.11.15）。難燃性大規模建造物では区画の独立性から否定の余地。
  2. 焼損＝独立燃焼説：火が媒介物を離れ目的物が独立燃焼に達した時に焼損＝既遂（最判昭23.11.2）。
  3. 故意の認識対象：客体が現住建造物（の一部）であることの認識で足り、現住区画への延焼の認識は不要。
  4. 論点コア：一体性肯定・独立燃焼・故意の三点同時充足で現住建造物等放火既遂が成立する型。

📚 根拠条文・判例

→ 刑法108条（現住建造物等放火罪） → 最判昭23.11.2（焼損＝独立燃焼説） → 大判昭9.11.15（建造物の一体性・棟割長屋）

一体性（木造＝肯定）・焼損（板壁の独立燃焼）・故意（現住部分の認識）の3論点がいずれも肯定方向に揃い、現住建造物等放火既遂が端的に成立する。

🎯 統合解説

📜 記述原文居住者全員を殺害した後、証拠隠滅のため家屋を焼失させようと室内の布団に放火したが、布団を焼損した時点で隣家の住民に消し止められた。→ 甲に非現住建造物等放火未遂罪が成立する。

💡 THE GIST○。居住者を皆殺しにした後の家屋は非現住建造物にすぎず、しかも燃えたのは建造物の一部でない布団だけ。建造物が独立燃焼に至らない以上、非現住建造物等放火の「未遂」が正解。

1罪名＝非現住現住建造物（108条）は「現に人が住居に使用し又は現に人がいる」ことを要する。家人を皆殺しにした後の家屋は住居使用者も在室者もおらず、判例は現住建造物等放火罪の成立を否定する（大判大6.4.13）。成立し得るのは非現住建造物等放火罪（109条）。

2客体の範囲放火罪の既遂は客体たる建造物の独立燃焼を要する。建造物を毀損せず取り外せる物は建造物の一部に当たらない（最判昭25.12.14）。布団は容易に取り外せる動産で建造物の一部ではない。

3既遂時期＝未遂布団が焼損しても建造物そのものは独立燃焼に至っておらず、消し止められた本事例では既遂に達していない。よって非現住建造物等放火の未遂（109条・112条）にとどまる。

💭 INTUITION住んでいた人がいなくなれば家のランクは「現住→非現住」へ格下げ。さらに燃えたのが「持ち出せる布団」だけなら、家本体はまだ燃えていない＝ゴールテープ未通過の未遂。

  1. 現住性の喪失（皆殺し）：居住者全員殺害後の家屋は非現住建造物（大判大6.4.13）。成立し得るのは109条。
  2. 建造物の一部性：毀損せず取り外せる物（布団）は建造物の一部でない（最判昭25.12.14）→建造物未燃焼。
  3. 既遂時期：客体たる建造物が独立燃焼に至らなければ既遂とならず未遂にとどまる。
  4. 論点コア：罪名（皆殺し→非現住）と既遂時期（建造物の一部でない物の焼損は未遂）の二段で評価する型。

📚 根拠条文・判例

→ 大判大6.4.13（家人皆殺し後の放火＝非現住） → 最判昭25.12.14（建造物の一部性・取り外し可能物） → 刑法109条（非現住建造物等放火罪）

罪名は皆殺し→非現住（大判大6.4.13）、既遂否定は布団＝建造物の一部でない（最判昭25.12.14）。両判例を重ねると非現住建造物等放火の「未遂」が導かれる。

🎯 統合解説

📜 記述原文妻と二人で自宅に居住していた甲が、妻の旅行中に火災保険金目的で自宅に放火して全焼させ、予期していなかった隣家にも延焼した。→ 甲に延焼罪が成立する。

💡 THE GIST×。妻が住む自宅は一時不在でも現住建造物（108条）のままで、甲は現住建造物等放火既遂罪。延焼罪（111条1項）は自己所有の非現住・建造物等放火（109条2項・110条2項）を前提とする加重類型だから、108条が基本犯の本事例では成立しない。

1「人」＝犯人以外108条「現に人が住居に使用し」の「人」は犯人以外の者を意味し犯人の家族を含む（最判昭32.6.21）。妻は犯人甲以外の者で、自宅は妻が住居に使用する建造物である。

2現住性＝使用形態現住性の喪失は人の起居の場所として日常使用するという建造物の「使用形態の変更」の有無で判断される（最決平9.10.21）。旅行中の妻が帰宅すれば再び住居として使用するから使用形態の変更はなく、現住性は失われない。→ 甲は現住建造物等放火既遂罪（108条）。

3延焼罪の前提延焼罪（111条1項）は109条2項又は110条2項（自己所有の非現住建造物・建造物等）の罪を犯し、よって108条・109条1項の物に延焼させた場合の加重類型。基本犯が108条の本事例は前提条文を欠き延焼罪は成立しない。隣家への延焼は108条放火既遂の中で評価される。

💭 INTUITION家族が留守でも家は「人の住む家」のまま。延焼罪という上乗せ規定は「自分の物（非現住）を燃やしたら他人の家に燃え移った」場面専用のはしご。最初から他人も住む現住の家を燃やした人は、このはしごには乗らない。

  1. 108条の「人」：犯人以外の者を指し犯人の家族を含む。家族が住む自宅は一時不在でも現住建造物（最判昭32.6.21）。
  2. 現住性の喪失基準：起居の場所として日常使用する使用形態の変更の有無で判断する（最決平9.10.21）。一時不在では失われない。
  3. 延焼罪の前提条文：111条1項は109条2項・110条2項（自己所有・非現住）を基本犯とする加重類型。基本犯が108条のときは前提を欠く。
  4. 論点コア：一時不在＝非現住と短絡し、かつ延焼罪の前提条文を確認しないと誤るひっかけの型。

3カーテン→未遂窓のカーテンはフックに掛けるだけで容易に取り外せる典型的な動産で建造物の一部でない。カーテンの焼損は「建造物の焼損」ではなく、躯体が独立燃焼に至る前に消し止められた本事例では既遂に達していない。成立し得るのは未遂であって既遂ではない。

💭 INTUITIONカーテンは「家の備品」であって「家そのもの」ではない。備品が燃えても建物本体に火が回るまでは放火のゴールテープは切れていない＝既遂手前の未遂。

  1. 建造物の意義：放火罪の建造物＝建造物損壊罪の客体と同一。容易に取り外せる物は建造物の一部でない（大判大8.5.13／最判昭25.12.14）。
  2. カーテンの位置づけ：容易に取り外せる物で建造物の一部でない→焼損しても建造物は未燃焼。
  3. 既遂時期：客体性（現住建造物か）と既遂時期（建造物が独立燃焼したか）は別問題。建造物の一部でない物の焼損では既遂とならない。
  4. 論点コア：客体が現住建造物でも、燃えたのが建造物の一部でないなら既遂に届かないという既遂時期の型（当該結論と同型）。

🔗 横断 ── 放火罪「焼損（既遂）」と「客体（現住性）」の二層構造

放火罪の成否は、つねに〈①客体が何か（現住／非現住）〉と〈②建造物そのものが独立燃焼したか（既遂時期）〉の二層で決まる。本問は4事例がこの二層のどこで決着するかを問う。客体論（記述ア・ウ）と既遂時期論（記述イ・エ）は層が違うので混同しないことが要点。

層第1層：客体（現住性・一体性）第2層：既遂時期（焼損）
判定基準一体性（外観上一個か）／「人」は犯人以外で家族を含む／使用形態の変更の有無独立燃焼説（火が媒介物を離れ目的物が独立燃焼）／燃えた物が建造物の一部か
主な判例大判昭9.11.15・最判昭32.6.21・最決平9.10.21・大判大6.4.13最判昭23.11.2・最判昭25.12.14・大判大8.5.13

決め手客体が現住建造物でも、燃えたのが建造物の一部でなければ既遂に届かない（記述エ）。逆に客体が非現住でも建造物が独立燃焼すれば既遂になる。客体の論点と既遂時期の論点を別々に詰めるのが鉄則。

THROUGH-LINE放火罪はどの事例でも「何を客体に放火したか」と「その建造物が独立燃焼したか」を二段で問えば足りる。延焼罪（111条1項）はこれに「基本犯が109条2項・110条2項か」という前提条文チェックが上乗せされる。
