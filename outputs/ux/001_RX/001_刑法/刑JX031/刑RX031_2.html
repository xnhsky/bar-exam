単純遺棄罪（217条）の「遺棄」の意義
― 置き去り（不作為形態）を含むか

出典: 刑JX031（たまたま通りかかった丙が要扶助者を置き去りにした事案）

問題の所在

保護責任のない者（丙）が要扶助者をその場に放置して立ち去った（置き去り＝不作為形態）。まず保護責任者遺棄罪（218条）は保護責任がなく不成立だが、単純遺棄罪（217条）の「遺棄」に置き去り（不作為形態）が含まれるかが問題となる。

規範（暗記対象）

217条の「遺棄」＝移置（作為形態）のみ／狭義説（判例・通説）
  1. 「遺棄」には、要扶助者を保護のない場所へ移転させる移置（作為形態）と、その場に放置して立ち去る置き去り（不作為形態）がある。
  2. 217条は保護責任を要しない非身分犯であるところ、保護責任のない者には不作為を処罰するための前提となる作為義務を観念できない。
  3. ゆえに217条の「遺棄」は作為形態である移置のみを指し、置き去り（不作為形態）を含まない（最判昭34.7.24）。
  4. 不作為形態（置去り・不保護）は、作為義務を負う保護責任者についてのみ、身分犯である218条で処罰される。

理由づけ

218条は「保護する責任のある者」という身分（＝作為義務）を要する真正身分犯だから、不作為形態（置去り・不保護）も処罰しうる。これに対し217条は保護責任を要しない非身分犯であり、その主体に作為義務はない。作為義務のない者の不作為を処罰すれば不真正不作為犯の理論と矛盾する。両条の体系的相違が、217条の「遺棄」を移置に限定する解釈を正当化する。

あてはめの型

  1. 218条：保護責任＝作為義務（判例通説）。丙に先行行為・排他的支配・法令上の義務なし → 保護責任なし → 218条不成立。
  2. 217条客体：72歳・認知症のA → 「扶助を必要とする者」に該当。
  3. 217条「遺棄」：非身分犯ゆえ作為義務なし → 移置（作為形態）のみ。
  4. あてはめ：丙は「何もすることなくその場を離れた」＝置き去り（不作為形態）→ 「遺棄」不該当。
  5. 結論：218条・217条いずれも不成立 → 丙は不可罰。

217条 / 218条 の比較

観点217条（単純遺棄）218条（保護責任者遺棄）
性質非身分犯真正身分犯
主体制限なし（保護責任不要）保護する責任のある者
「遺棄」移置（作為）のみ移置＋置去り／不保護（不作為）
本問丙置き去り→不該当・不成立保護責任なし→不成立

関連判例・条文

最判昭34.7.24（217条「遺棄」は移置のみ・置き去りを含まない）／大判大4.5.21（移置＋置去り双方を含むとした古典的広義説・対比）

規範チェック（○×）

単純遺棄罪（217条）の「遺棄」は移置（作為形態）のみを指し、置き去り（不作為形態）を含まない。
217条の「遺棄」が移置に限定されるのは、217条が保護責任者という身分を要する身分犯だからである。
たまたま通りかかっただけで先行行為も保護の引受けもない者には、保護責任（作為義務）が認められず、保護責任者遺棄罪（218条）は成立しない。
丙が要扶助者をその場に放置して立ち去った行為は「移置」に当たり、217条の「遺棄」として処罰される。
