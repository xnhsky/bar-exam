自己物の取戻しと
自救行為による違法性阻却

問題の所在

自己の財物が他人の違法な占有下にある場合、占有説によれば取戻し行為は窃盗罪の構成要件に該当する（→保護法益論カード）。もっとも、自己物を取り戻すだけの行為まで一律に処罰するのは酷である。そこで、自救行為として違法性が阻却されないかが問題となる。

規範（暗記対象）

自救行為による違法性阻却（最決平元.7.7 参照）

自救行為とは、正当防衛を認めるだけの侵害の急迫性・現在性は存在しないが、国家機関の救済を待っていては権利の回復が困難になる場合に、自ら実力により救済を図る行為をいう。自力救済禁止の趣旨を没却するおそれがあるから、次の要件を充たす場合に限り違法性が阻却される。

  1. ① 法益に対する違法な侵害があること
  2. ② 国家機関に救済を委ねれば回復が事実上不可能・著しく困難な明白な事態
  3. ③ 社会観念上相当な侵害回復行為であること（手段の相当性）
  4. ④ 自救の意思（専ら権利回復目的）

要するに、社会通念上、被害者に受忍を求める限度を超えない場合に限り違法性が阻却される。

理由づけ

自力救済は原則として禁止される（社会の平穏・秩序）。しかし、急迫不正の侵害は去っており正当防衛は使えず、かつ司法手続を待てば権利回復が著しく困難な場面では、私人の実力行使を一切認めないと権利者保護に欠ける。そこで急迫性に代えて厳格な要件を課し、受忍限度を超えない範囲で例外的に違法性阻却を認める。

あてはめの型（事実をゴリゴリ拾う）

  • ①違法な侵害 … 取戻し対象の占有がそもそも違法な占有か
  • ②回復困難 … 相手に返還を言い出せない事情／置き忘れの隙のような一過性の機会しかなかったか（国家機関の救済を待てない）
  • ③手段の相当性 … 有形力を行使していないか／別の法益（住居等）を侵害していないか／取り返した態様の穏当さ
  • ④自救の意思 … 専ら自己物の取戻しが目的か（腹いせ等の別目的でないか）

関連判例・条文

最決平元.7.7（百選Ⅱ26） … 自己物取戻し事案で占有説に立ちつつ違法性阻却の枠組を示した（「社会通念上受忍を求める限度」）

限界 … 他人の住居に侵入して取り戻す等、別の法益侵害を伴う場合は自救行為として違法性は阻却されない

条文 … 刑法235条（窃盗）／242条（自己の財物）／36条1項（正当防衛＝急迫性ありとの対比）

規範チェック（○×）

自救行為は、正当防衛と異なり、侵害の急迫性・現在性が存在しない場面で、国家機関の救済を待てないときに問題となる。
自己物の取戻しであれば、取り戻すと同時に相手への腹いせ・制裁を目的としていても、自救行為として違法性が阻却される。
自己物の取戻しが目的でありさえすれば、他人の住居に侵入して取り戻す行為も、手段を問わず自救行為として違法性が阻却される。
自救行為による違法性阻却の可否は、最終的に「社会通念上、被害者に受忍を求める限度を超えないか」という観点から判断される。
