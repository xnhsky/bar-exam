二重譲渡をめぐる詐欺罪（財産上の損害・三角詐欺の処分権限）

出典: 刑JX055　不動産二重譲渡における横領罪・詐欺罪（論点2・甲の罪責②）

問題の所在

甲はAへの売却を秘してB・C銀行と取引した。詐欺罪（246条1項）で問題となるのは、二重譲渡では第二譲受人が登記を備えれば対抗できる（民法177条）以上、原則として財産上の損害がないこと。例外的成立の可否と、被欺罔者（C銀行）と被害者（A）が異なる三角詐欺の成否が争点となる。

規範（暗記対象）

一般に、二重譲渡事案においては、第二譲受人は移転登記を備えれば第一譲受人に所有権の移転を対抗し得る（民法177条）から、登記を得れば財産上の損害はなく、原則として詐欺罪は成立しない。もっとも、買主が登記を得てもなお取引の目的を達し得ない特段の事情がある場合には、実質的にみて財産上の損害を認め、例外的に詐欺罪の成立を肯定し得る（理論構成としては欺罔行為の重要事項性・錯誤・財産的損害のいずれに引っ掛けてもよい）。
  1. 三角詐欺（被欺罔者と財産上の被害者とが異なる詐欺）が成立するためには、被欺罔者が被害者の財産を処分し得る権限又は地位を有することを要する（最判昭45.3.26）。
  2. 悪意者は錯誤に陥らないから、錯誤を招来するような欺罔行為が認められず、詐欺罪は成立し得ない。
最判昭45.3.26（三角詐欺・被欺罔者の処分権限を要する・百選II56）／民法177条（不動産物権変動の対抗要件）

理由づけ

詐欺罪は個別財産に対する罪であるが、実質的個別財産説によれば、交付の判断の基礎となる重要事項を偽り、被害者が取引目的を達し得ない場合に実質的な財産的損害を認める。三角詐欺で被欺罔者の処分権限を要求するのは、処分権限があって初めて被欺罔者の処分を被害者の財産処分と同視でき、被害者の意思に基づかない移転である窃盗罪との区別が明確になるからである。

あてはめの型

  1. Bへの売却：Aへの売却を秘した契約＝欺罔。もっとも原則は登記で対抗可（民177）→ 財産上の損害なし → 原則不成立
  2. 例外：Bは登記前に解除を申し入れている → Aの存在を知れば買わなかったと評価 → 特段の事情 → 例外的に財産上の損害を肯定 → 詐欺成立を認め得る（否定の結論も可）
  3. 1100万円の返還は詐欺成立（代金受領時に既遂）後の事情 → 成否に影響しない
  4. C銀行への二当事者間詐欺：知れば融資しなかった事情なし → 不成立（「金融機関だから」の深読みは問題文に根拠なく不可）
  5. Aを被害者とする三角詐欺：C銀行にAの財産処分権限なし（最判昭45.3.26）→ 不成立
  6. 乙への詐欺：乙は悪意で錯誤なし → 成立し得ない

関連判例・条文

条文　刑法246条1項（詐欺）／民法177条（不動産物権変動の対抗要件）

判例　最判昭45.3.26（三角詐欺の成立には被欺罔者が被害者の財産を処分し得る権限・地位を要する・百選II56）

規範チェック

Q1.　二重譲渡では第二譲受人は登記を備えれば対抗できるが、第二譲受人は常に財産上の損害を被るから、詐欺罪は当然に成立する。
Q2.　Bが甲A間の売買を知って契約を解除した事情は、登記を得てもなお取引目的を達し得ない特段の事情として、例外的に財産上の損害を基礎づけ得る。
Q3.　三角詐欺は被欺罔者と被害者が異なれば足り、被欺罔者が被害者の財産を処分し得る権限・地位を有することは必要でない。
Q4.　甲がBに1100万円を返還した事実は、Bに対する詐欺罪の成立を遡って否定し、同罪は成立しないことになる。
