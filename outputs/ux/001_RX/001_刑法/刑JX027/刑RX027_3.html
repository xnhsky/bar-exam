共同の注意義務の限界（過失共同正犯の否定と不可罰の二段論証）

出典: 刑JX027（過失の共同正犯・業務上過失致傷罪）／論点3・補助作業者丙の罪責

① 問題の所在

同じ建築会社の作業員でも、補助作業しかしていなかった者（丙）に業務上過失致傷罪が成立するか。共同正犯（→甲のカード）で認めた「共同の注意義務」がどこまで広がるか＝その限界が問われる。共同正犯と単独犯の両面から二段で切る。

② 規範（暗記対象）

【共同正犯否定の規範】共同の注意義務の不存在

共同の注意義務は無限定に広がるものではない。共同作業に従事していても、地位・役割が異なり、当該危険を共同して防止すべき関係にない者には共同の注意義務は認められず、過失の共同正犯は成立しない（信頼の原則も妥当）。

【単独犯否定の規範】固有の過失の不存在

単独犯が成立するには行為者固有の結果回避義務違反（過失）が必要。当該結果の原因が専ら他者の不注意にあり、本人の作業に注意義務違反が認められない場合は、単独犯も成立しない。

③ 理由づけ

  • 共同義務を限界づける理由：共同義務を無限定に広げれば、現場に居合わせただけの者まで一部実行全部責任を負い処罰範囲が不当に拡大する。地位・役割・要求行為の共通性で画する必要がある。
  • 信頼の原則：他の関与者（乙）が適切に行動することを信頼してよい立場の者は、その者の作業に伴う危険まで防止すべき注意義務を負わない。
  • 対照の意義：甲（肯定）と丙（否定）のコントラストで、共同義務の認定基準を「無限定でない」ものとして示せる。

④ あてはめの型

  1. 共同正犯の否定：丙は建築作業員になって日が浅く、木材を運ぶなど補助的作業のみ → 乙の高所作業に伴う落下の危険を共同して防止すべき地位になし → 共同の注意義務なし → 共同正犯否定。
  2. 単独犯の否定：瓦は「乙が注意を払って取り扱っていれば落下等の危険のない状態」で積まれていた → 丙が運び積む際の注意義務違反は認められず、落下の原因は専ら乙の取扱い → 固有の過失なし → 単独犯否定。
  3. 結論：丙は業務上過失致傷罪の罪責を負わない（不可罰）。

⑤ 関連判例・条文

条文：刑法211条前段（業務上過失致傷罪）／60条（共同正犯の否定）

判例：最決平28.7.12（明石歩道橋事件・地位役割の相違で共同義務を否定した枠組み）／札幌高判昭51.3.18（北大電気メス事件・信頼の原則による注意義務の限定）

⑥ 規範チェック（○×）

同じ建築会社で共同作業に従事している以上、補助作業しかしていない丙にも当然に共同の注意義務が認められる。
丙を不可罰とするには、共同正犯の否定（共同義務の不存在）に加えて、単独犯の否定（丙固有の過失の不存在）も論じる必要がある。
瓦を2階へ運び積み上げた丙には、その積み上げ行為自体に結果回避義務違反（固有の過失）が認められる。
