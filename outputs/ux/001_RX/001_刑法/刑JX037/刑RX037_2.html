死者の占有― 客観は生者・主観は死者と窃盗罪の成否

RX 論証カード ／ 出典: 刑JX037

問題の所在

暴行後に初めて財物奪取意思を生じた場合、暴行時に奪取意思がないため強盗は成立しない。残るは窃盗だが、本問は奪取時に被害者がなお生存していた一方、行為者は被害者を死亡したと誤信していた。客観は生者の占有侵害だが、主観は「死体からの奪取」。この主観のもとで窃盗の故意を認めうるかが死者の占有として問題となる。

規範（暗記対象）

死者の占有（最判昭41.4.8）

死者には財物に対する占有の事実も意思も認められないのが原則である。しかし、人を死亡させた者がその直後に同一の機会・場所で財物を奪取する場合には、被害者が生前有していた財物の所持は、その死亡直後においてもなお継続して保護するのが法の目的にかなう。

したがって、被害者から財物の占有を離脱させた自己の行為を利用して財物を奪取した一連の行為を全体的に考察すれば、他人の財物に対する所持を侵害したものといえ、占有離脱物横領ではなく窃盗罪を構成する。

強盗の奪取意思の発生時期

強盗罪（236条1項）は暴行・脅迫を手段として財物を強取する点に本質があるから、暴行・脅迫の時点で財物奪取の意思を要する。奪取意思が暴行後に生じた場合は強取とはいえず強盗は成立しない。

理由づけ

被害者本人が死亡（と認識）した直後であっても、生前の占有を保護の空白に置くのは法の目的に反する。そこで一連の行為を全体的に考察し、生前の占有侵害として窃盗の保護を及ぼす。本問の特殊性は、客観的には乙が生存しており端的に生者の占有侵害＝窃盗の客観的構成要件に該当する点。問題は主観面に尽き、甲の認識（死亡直後の奪取）を基準としても昭41判例の趣旨で窃盗の故意が肯定される。よって客観・主観いずれの面からも窃盗が成立する。

あてはめの型

  1. 強盗の成否を先に切る（暴行時に奪取意思なし → 強盗・強盗致死とも不成立）
  2. 客観面 → 奪取時、被害者は生存 → 生者の占有を侵害 → 窃盗の客観的構成要件該当
  3. 主観面 → 行為者は死亡を誤信 → 死者の占有が問題
  4. 昭41判例の趣旨（生前占有の死亡直後の継続的保護・一連の行為の全体的考察）を援用 → 甲の認識を基準としても窃盗の故意あり
  5. 不法領得の意思を肯定（金策目的＝権利者排除＋利用処分の意思）
  6. 窃盗罪（235条）成立／傷害罪とは別個の行為で併合罪（45条前段）

関連判例・条文

最判昭41.4.8（百選Ⅱ29）… 殺害直後に現場で財物奪取。生前の占有を死亡直後もなお継続保護し、一連の行為を全体的に考察して窃盗罪を肯定（占有離脱物横領ではない）

三類型の整理 … ①行為者自身が殺害→直後に奪取＝窃盗（昭41）／②第三者殺害後に無関係者が奪取＝占有離脱物横領／③当初から奪取目的で殺害＝強盗（致死）

条文 … 刑法235条（窃盗）／236条1項（強盗）／38条1項（故意）／254条（占有離脱物横領）／45条前段（併合罪）

規範チェック

暴行を加えた後に初めて財物奪取の意思を生じ、その財物を奪取した場合でも、結果的に暴行と奪取があれば強盗罪が成立する。
人を死亡させた者がその直後に同一の現場で財物を奪取した場合、被害者が生前有していた占有がなお継続的に保護され、一連の行為を全体的に考察して窃盗罪が成立する。
行為者が被害者を死亡したと誤信して財物を奪取したが、奪取時に被害者がなお生存していた場合、客観面の検討で死者の占有を論じることになる。
行為者が死者からの奪取と認識していた以上、客観的に被害者が生存していても、成立しうるのは占有離脱物横領罪（254条）にとどまる。
