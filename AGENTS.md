# AGENTS.md instructions

私はPC初心者で、コードはほとんど書けません。
Codexは作業だけで終わらず、初心者の私が次に何をすればいいかまで日本語で丁寧に案内してください。

## 作業前

- 複数ステップになりそうな場合は、最初に簡単な作業計画を出してください。
- 危険な操作、削除、上書き、依存関係の追加、デプロイ、本番環境変更は、勝手に進めず事前に確認してください。
- Git操作のうち、このリポジトリ内の通常作業で必要になるコミットとプッシュはユーザーから包括委任済みです。作業完了後、関連ファイルだけを確認してコミットし、originへプッシュしてください。
- ただし、履歴改変、強制push、リセット、ブランチ削除、タグ操作、リリース、PR作成、デプロイ、本番環境変更は事前確認してください。

## 作業中

- コマンドを出す場合は、どこで実行するのか、何をするコマンドなのか、成功するとどう見えるのかを説明してください。
- 専門用語はできるだけ避け、必要な場合は短く説明してください。
- なるべく安全で小さい変更にしてください。

## 作業後の報告形式

作業後は必ず、日本語で次の形式でまとめてください。

1. 完了したこと
2. 変更したファイル
3. 確認したこと / 実行したテスト
4. 残タスク
5. 次に私がやること
