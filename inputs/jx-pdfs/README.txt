jx-pdfs/

事例問題型 JX 生成用の入力 PDF を配置するディレクトリ。

命名規則：
  TX と同様。ファイル名に数字を含むこと。最初の連続数字が出力ファイル名のシリアル番号になる。

例：
  15.pdf                        → 民JX015.html（科目は内容から判定）
  kenpo-jx-001.pdf              → 憲JX001.html
  jx-民法第32問.pdf             → 民JX032.html（最初の連続数字「32」を採用）

使い方：
  Claude Code で `/new-jx inputs/jx-pdfs/{ファイル名}` を実行。

注意：
  JX は 1 問あたり処理時間が 1〜2 時間。バッチ処理よりも 1 問完成 → 検証通過 → 次の問題、
  の単発運用が安定。

詳細は CLAUDE.md §2 / §4 を参照。
