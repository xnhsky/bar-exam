# LEXIA 過去問周回正典指南書

作成日: 2026-06-28

## 0. 結論

LEXIA と生成物を主戦場にしてよい。

ただし、目標は「基本書を読まなくても済むほど、過去問から抽出した条文・判例・出題意図・答案構成を高速再現できる状態」にすることであり、生成物を単なる長文解説や暗記ノートにしてはならない。

基本書は通読対象ではなく、最後の検算用辞書である。読むとしても、疑問点を確認するために短時間だけ使い、その理解は JX / TX-LEX / ARIADNE / RX へ吸収して、すぐ LEXIA 周回へ戻る。

## 1. 学習思想

### 採用する思想

- 過去問こそ最高の教科書である。
- 短期合格では、過去問周回の質と速度が最重要である。
- 司法試験・予備試験は、条文と判例を問題文の事実にぶつけるゲームである。
- 条文の制度趣旨・保護法益は、短答でも論文でも大前提である。
- 出題意図は極めて重要である。
- SM2 は、忘却対策ではなく「答案再現・短答反射」を作るために使う。
- 移動中学習が多い前提で、音声化・想起・短いストーリー理解に耐える教材にする。

### 捨てる思想

- 基本書通読で安心する。
- 学説を深掘りして理解した気になる。
- 長い解説を読めば合格答案が書けると思う。
- 正誤だけ覚えて短答力が上がったと思う。
- 論証を丸暗記すれば論文答案が書けると思う。

## 2. 役割分担

### JX

JX は母艦である。基本書代替の知識層として置く。

ただし、毎日読む主教材ではない。ARIADNE / RX / TX-LEX で詰まった時に戻る参照先であり、最終的には JX の内容を周回可能なカードや答案構成へ変換する。

JX に必要なもの:

- 問題文の構造分析
- 条文の位置付け
- 判例射程
- 制度趣旨・保護法益
- 答案上の使い方
- 失点例
- 参照条文・判例・論点の整理

JX に不要なもの:

- 過度な学説博覧会
- 基本書的な周辺論点展開
- 本問で使わない知識の長い補足

### TX-LEX

TX-LEX は短答の主戦場である。

目的は、条文・判例・誤肢処理を反射化すること。正答根拠を読む教材ではなく、誤った一般化を瞬時に切る教材にする。

各肢に必要な要素:

- 条文のどの文言か
- 制度趣旨・保護法益は何か
- 判例の射程はどこまでか
- 誤肢ならどこで切るか
- 似た肢に転用するときの注意点

TX-LEX の理想形:

- 1肢1ストーリー
- 1肢1切断点
- 1肢1条文・判例接続
- 長文解説ではなく、移動中に思い出せる短い因果で書く

悪い TX-LEX:

- 「これは誤り」だけで終わる。
- 条文文言に戻らない。
- 判例の事案射程を示さない。
- 誤肢のどの語が危険かを示さない。

### ARIADNE

ARIADNE は論文の主戦場である。

目的は、移動中でも答案構成を脳内再現できる状態にすること。解説ファイルではなく、答案再現装置である。

ARIADNE に必須の層:

1. 出題意図カード
2. 条文・保護法益カード
3. 30秒答案構成
4. 2分答案構成
5. 本番答案構成
6. 事実評価マトリクス
7. A答案 / B答案 / 危険答案の分岐
8. 想起ドリル
9. 模範答案
10. 駆け込み寺ストーリー解説

ARIADNE で厚くする場所:

- 出題意図
- 条文選択
- 保護法益・制度趣旨
- 問題文事実の拾い方
- 事実から評価語への変換
- 採点者が点を入れる箇所
- 書きすぎ防止
- 危険答案の回避

ARIADNE で厚くしない場所:

- 学説紹介
- 周辺論点
- 基本書的な網羅解説
- 本問で答案に書かない理論説明

ARIADNE の想起クイズ / RX 接続:

ARIADNE の想起クイズは、短答の一問一答ではない。答案構成前に「次に答案で何を書くか」を強制する論文用一問一答である。

想起クイズで問うもの:

- この事案で最初に厚く論じる人物・罪名・山場は何か。
- その争点は条文のどの要件に乗るか。
- 採点者が見たい決定的事実はどれか。
- その事実をどの評価語へ変換するか。
- A答案とB答案の差はどこか。
- 危険答案は何を盛りすぎるか。
- 最後に一文で切る論点は何か。

RX への接続ルール:

- ARIADNE が主役、RX は補助弾倉である。
- 想起クイズに答えられなかった箇所だけ RX に飛ぶ。
- RX で見るのは、規範・理由・答案フレーズ・あてはめタグに限る。
- RX を読み込んだら、必ず ARIADNE に戻って答案構成を再現する。
- RX を連続で読み続けない。RX は迷った時の再装填であり、主教材ではない。

### RX

RX は論証の補助弾倉である。

主役にしない。ARIADNE で詰まった時に、規範・理由・答案フレーズを短く再装填するためのカードにする。

RX の基本構造:

- 発動条件
- 規範
- 理由づけ
- あてはめタグ
- 答案フレーズ
- ひっかけ命題

RX に不要なもの:

- 詳細な学説対立
- 長い判例解説
- JX の代替になるほどの厚い説明

### TREE

TREE は地図である。

迷子防止と直前確認用であり、読み込む教材ではない。全体像・論点順序・失点防止ワードを即座に確認できればよい。

TREE の各葉に必要なもの:

- 固有の事実評価
- 失点例
- 評価語
- 答案上の置き場所

TREE に不要なもの:

- 反復的な一般文
- 全葉に同じ説明を貼ること
- 詳しい解説

## 3. 基本書の扱い

基本書は買ってもよい。

しかし、通読してはいけない。通読は安心感を生むが、短期合格の速度を落とす危険が高い。

基本書の使い方:

- どうしても理解できない時にだけ見る。
- 1回10分までを目安にする。
- 読んだ内容をそのまま増やさない。
- 必ず JX / ARIADNE / RX / TX-LEX のいずれかに圧縮して戻す。
- 基本書を周回教材にしない。

基本書を見るべき場面:

- 条文の制度趣旨が生成物だけでは見えない。
- 判例射程が不安。
- 似た論点の区別がどうしても崩れる。
- 生成物の説明に違和感がある。

基本書を見てはいけない場面:

- なんとなく不安。
- 今日の周回から逃げたい。
- 完璧に理解してから進みたい。
- 周辺論点もついでに押さえたい。

## 4. 駆け込み寺ストーリー解説

ストーリー解説は載せてよい。

ただし、主ルートに混ぜてはいけない。周回速度を落とすからである。

ストーリー解説は、ARIADNE や TX-LEX の折りたたみ層に置く。開く条件は「想起できない」「答案構成が崩れる」「条文と事実の接続が見えない」時だけである。

ストーリー解説の役割:

- 基本書の代替
- 移動中に頭へ入る理解補助
- 出題意図と条文判例の接続
- 事実評価の流れを一本の物語にする

ストーリー解説の構造:

1. この問題は何を聞いているか
2. なぜこの条文が出るか
3. 制度趣旨・保護法益は何か
4. 判例は何のズレを処理したか
5. 問題文の事実がどの要件に刺さるか
6. 答案ではどう短く書くか

ストーリー解説の禁止事項:

- 本問で使わない学説を広げる。
- 周辺論点を増やす。
- 結局何を書けばよいか不明な長文にする。
- 何度も読ませる前提にする。

## 5. SM2 用カード設計

SM2 で回すカードは、認識カードではなく再現カードにする。

悪いカード:

- 「この記述は正しいか」
- 「判例名は何か」
- 「論点名は何か」

良いカード:

- 「次に答案で何を書くか」
- 「この事実はどの要件に使うか」
- 「この事実評価の評価語は何か」
- 「ここで条文のどの文言に戻るか」
- 「誤肢のどの語で切るか」
- 「A答案と危険答案の差は何か」

ARIADNE の想起カード例:

- 甲で最初に検討し、すぐ否定する犯罪は何か。
- 相続利益を否定する評価語は何か。
- 甲の実行の着手で拾う三つの事実は何か。
- 乙では故意と因果関係のどちらを先に処理するか。
- 投棄行為の介在を切断しない理由は何か。
- 過失致死を別罪にしない理由は何か。

TX-LEX の想起カード例:

- この肢は条文文言のどこをずらしているか。
- この判例知識は事案射程を広げすぎていないか。
- 保護法益から見ると、この肢はなぜ危険か。
- 似た肢が出たら、どの語を見て切るか。

## 7. 新規生成ロードマップ

新規生成は、刑法で固めた正典思想をそのまま量産するのではなく、各科目の答案構造へ変換しながら進める。

推奨生成順序:

1. 刑法
2. 刑事訴訟法
3. 民法
4. 商法
5. 民事訴訟法
6. 憲法
7. 行政法

この順序を採る理由:

- 刑法から刑事訴訟法へ進むことで、刑事系の事実評価・条文判例処理・捜査公判への接続を切らさない。
- 民法を先に置くことで、商法・民事訴訟法の前提となる権利義務、要件効果、損害、抗弁構造を固める。
- 商法を民法の直後に置くことで、会社法・商行為を民法の応用として処理しやすくする。
- 民事訴訟法を商法の後に置き、民事系の実体法から手続法へ移る。
- 憲法を行政法より前に置く。憲法を最後にすると、権利選択、制約、審査基準、事実評価という公法答案の型が育つのが遅れる。
- 行政法は最後に置く。憲法で公法系の型を作ったうえで、処分性、原告適格、裁量、手続、救済選択へ進む。

厳守すること:

- 刑法の見た目や文言を他科目へ横流ししない。
- 共通思想は維持するが、切断点の言語は科目ごとに変える。
- 新規生成時から TX-LEX / ARIADNE / RX / JX / TREE の役割分担を固定する。
- 各科目セッションは、自分の科目フォルダだけを触る。
- 統括セッションで共通正典・科目別アダプタ・preflight を管理する。

科目別アダプタの軸:

- 刑事訴訟法: 強制処分、令状、違法収集証拠、自白、伝聞、訴因。
- 民法: 要件事実、効果、抗弁、再抗弁、利益衡量、判例射程。
- 商法: 条文操作、会社機関、責任追及、手続瑕疵、利益相反。
- 民事訴訟法: 訴訟物、既判力、弁論主義、処分権主義、証明責任。
- 憲法: 権利性、制約、審査基準、目的手段審査、事実評価。
- 行政法: 処分性、原告適格、裁量、手続、国家賠償、救済選択。

## 8. 刑JX019 への適用例

刑JX019 の出題意図:

- 甲は、予定より早く結果が発生した場面で、実行の着手を前倒しできるかを問う。
- 乙は、予定より遅く結果が発生した場面で、自己の介在行為を経ても因果関係をつなげられるかを問う。
- 共通して、因果関係の錯誤が故意を阻却しないことを問う。
- 甲の山は実行の着手であり、乙の山は因果関係である。

刑JX019 の30秒構成:

甲は、相続利益を財産上の利益に当たらないとして強盗殺人を否定し、殺人罪へ進む。第1行為は第2行為を確実かつ容易にするため必要不可欠で、約5km・約10分、障害事情もないから実行の着手を肯定する。刺殺予定と窒息死のズレは因果関係の錯誤にとどまり、故意を阻却しない。

乙は、絞首行為と砂末吸引死との因果関係を、危険の現実化で肯定する。投棄は発覚防止の事後処分で異常な介在ではない。頸部圧迫死の認識と砂末吸引死の現実のズレは故意を阻却せず、過失致死は殺人既遂に吸収される。

刑JX019 の駆け込み寺ストーリー:

甲は、予定より早く人が死んだ。まだ山中で刺していないのに、準備段階の強打・猿ぐつわ・トランク閉込めでAが窒息死している。だから問題は、「この段階でも殺人の実行に着手したといえるか」である。鍵は、第1行為が第2行為を確実かつ容易にするため必要不可欠で、時間的場所的にも接着し、障害もないこと。ここで着手を前倒しできれば、一連の殺人行為への認識として故意をつなげられる。

乙は、予定より遅く人が死んだ。首を絞めた時点で死んだと思ったが、実際にはその後の砂上放置で砂を吸って死んでいる。だから問題は、「首絞めと死亡結果を因果関係でつなげるか」である。鍵は、投棄が偶然の外部事情ではなく、発覚防止の事後処分として殺害行為に通常随伴すること。危険の現実化で因果関係をつなげたら、死因のズレは因果関係の錯誤として故意を阻却しない。

## 9. 最終ルール

深掘りではなく、出題意図に沿った高速再現。

基本書ではなく、過去問生成物。

理解のためのストーリーは許す。ただし折りたたむ。

SM2 は暗記ではなく、答案再現と短答反射のために使う。

LEXIA が本丸。基本書は保険。ストーリー解説は駆け込み寺。

## 10. 見た目・体裁・正典の扱い

見た目は飾りではない。周回速度、記憶定着、教材モードの切替を支える学習機能である。

TX-LEX / JX / ARIADNE / RX / TREE は、それぞれの配色・フォント・余白・カード感・ラベルの質感によって、学習者の脳内モードを切り替える。したがって、内容改善のために正典の見た目を劣化させてはならない。

正典改修時に守るもの:

- 正典の配色体系
- 正典のフォント体系
- 見出し、カード、ラベル、余白の質感
- TX / JX / ARIADNE / RX / TREE の役割差
- スマホ・移動中学習での読みやすさ
- 印刷・保存しても破綻しない構造
- validator 合格だけで終わらない誌面品質

ただし、正典は聖域ではない。

正典の見た目や構成が、学習効率・可読性・誤答防止・答案再現に反する場合は、第三者レビューとして改善候補に挙げる。Claude と作った既存設計は尊重するが、合格に反する甘さは残さない。

第三者レビューで見る観点:

- 情報密度が高すぎて移動中に読めない。
- 色や装飾が意味と対応していない。
- 見出しは美しいが、想起すべき問いが弱い。
- カードが多いだけで、SM2 向きの再現カードになっていない。
- 模範答案は美しいが、A答案 / B答案 / 危険答案の差が見えない。
- ストーリー解説が気持ちよく読めるだけで、答案に戻らない。
- TREE が見栄え重視で、直前確認ワードになっていない。
- RX が論証カードではなく小型 JX になっている。
- TX-LEX が短答の切断点ではなく、単なる解説になっている。

合言葉:

内容は試験委員目線。

構造は過去問周回目線。

見た目は正典厳守。

ただし、正典は尊重するが聖域にはしない。

美しさを壊さず、甘さは壊す。
