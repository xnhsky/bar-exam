## 3. 論点核心

横領罪（252条1項）の客体「自己の占有する他人の物」要件を、5 つの場面で多角的に検討する判例典型問題。

各記述の論点と判例:
- **ア**: 不動産二重売買（甲所有→乙売却→乙登記未完了→丙売却・丙登記完了）→ **成立する**（最判昭30.12.26）。所有名義が売主に残存する間、売主はその不動産を占有する者として横領罪の主体となる。
- **イ**: 所有権留保付き動産売買での担保提供（24回月賦・3回支払後）→ **成立する**（最決昭55.7.15）。代金未完済の段階で売主に所有権が留保されており、買主にとって自動車は「他人の物」。
- **ウ**: 盗品売却代金の着服 → **成立する**（最判昭36.10.10）。252条1項の「他人の物」は、給付者が民法上犯人に対し返還請求し得るかを要件としない。
- **エ**: 抵当権設定仮登記（他人所有建物・不実の仮登記）→ **成立する**（最決平21.3.26・重判平21刑6事件）。仮登記による権利の優先的主張可能性の確保が、不法領得の意思の実現として横領罪成立を肯定。
- **オ**: 自己所有不動産への二重抵当権設定（乙のため設定→丙のため設定・登記完了）→ **成立しない**（最判昭31.12.7・百選II70事件）。本件不動産は甲所有であり、横領罪の客体「他人の物」に当たらない。乙の一番抵当権を後順位に追いやった行為は背任罪（247条）を構成する。

## 4. theory_deep_dive 採否：**必須**

論点核心が **横領罪と背任罪の区別**（特にオ肢）にあるため、§17-ter 学説対立 deep-dive が必須。

- **主流説（major）**: 客体限定説（最判昭31.12.7・判例実務）
  - 横領罪の客体は「自己の占有する他人の物」に厳格に限定。自己所有不動産への二重抵当権設定は客体性を欠き、横領罪不成立・背任罪成立。
- **対立説（minor）**: 客体拡張説（一部学説）
  - 「他人の物」を実質的に解し、他人の権利の対象となっている自己所有物も客体に含めるべきとする立場。判例は採用せず。
