## 5. 対立軸 SVG（任意・推奨形式）

複雑な学説対立では 2 次元プロット推奨：

- 横軸：要件論 ↔ 効果論 / 形式論理 ↔ 実質的妥当性 等
- 縦軸：根拠の重み付け軸
- 各学説を 2D 平面上の点（rect or circle）として配置
- 軸ラベル + 各学説名
- viewBox 800×400 推奨

CSS class：`.theory-axis-svg` 配下に必要なノードクラスを定義（Task 11 で詳細確定）。
