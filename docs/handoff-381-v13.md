> **Rストリーム（`tx-v13-runner.ps1 -Regen`＝PDF→v13）で v13.1.0 化を完了**した（記述数 N=8 を維持・
> 8カード/8 ox-row/#stmt-1..8・COMBOエンジン・nb-badge・横串trap・全ゲート PASS）。
> 標準手順は `docs/tx-v13-migration-targets.md`「特殊型の v13.1.0 化・標準手順」を参照。
> 以下は当時の設計メモ（歴史的参照）。手作業の個別設計は不要になった。

> リモートセッションからの引き継ぎ。381 は valid v11 のまま据置中（壊れていない・全ゲート PASS）。
> v13.1.0 LOOP-CARD 化はローカルで行う。以下は現状把握・注意点・手順。

## 1. なぜリモートで自動変換しなかったか（重要）

- **8穴埋めの学生対話・学説型**問題（`ox-row` 8・`choice-section` 5・`data-answer-type=ox-grid`・
  `data-correct-value=○○○○○○○○`）。問題文は「学生A：判例は…（①）し、その場合の名義人を（②）…」の
  対話穴埋め。論点＝**写真コピーの文書性／補助公務員の作成権限／有形偽造と無形偽造の区別／間接正犯**。
- 決定論ラッパー `scripts/tx-lex-v11-to-v13.py` は **5記述・客体三分の体系マップ**を前提にしており、
  8記述・学説型とは**構造的に非互換**（無理に通すと体系マップ／カード構造が破綻する）。
- **PDF が `inputs/000_TX/001_刑法/` に無い（Drive のみ）**ため、Rストリーム（`tx-v13-runner.ps1 -Regen`＝
  PDF→v13）でも再生成できない。

## 2. ローカル作業手順（案）

1. **Drive から PDF を回収**：`マイドライブ / 1 TX_短答 / 001_刑法 / 抽出PDF` の 381（R7-14 公文書偽造罪等・
   共通問題）を `inputs/000_TX/001_刑法/381.pdf` へ置く。
2. **PDF と現行 v11 の 8記述を照合**（現行 `_lex` は内容自体は妥当。下記 §3 の 8記述を参照）。
3. **v13 化の設計判断（学説型のため定型に乗らない）**：
   - 体系マップ（`.tx-sysmap-svg`）を **5節客体モデルで無理に描かない**。この問題の骨格は
     「①コピーの文書性 → ②名義人 → ③反対説」「④補助公務員の権限 → ⑤反対説」「⑥無形偽造(156条) →
     ⑦補助公務員による間接的無形偽造 → ⑧非公務員の間接正犯(157条・特別規定)」の**3系統×学説対立**。
     体系マップは「有形偽造 vs 無形偽造」「作成権限の所在」を軸にした**問題固有の作図**が要る。
   - あるいは、**5記述前提の LOOP-CARD に載らない問題は当面 v12.2.1（GENESIS-CORE）系で仕上げる**判断も可。
     v13 は 5記述・客体型に最適化されており、8穴埋め学説は v12.2.1 の方が収まる可能性がある。
4. 生成後は `validate-tx-core.py`（G1〜G53）／`check-tx-lex-engine.py`／`check-duplicates.py` を通す。
   v13 化するなら G50/G51/G52（各カードに synthesis/POINT/BASIS/**横串trap**、BASIS 非空）を満たすこと。
5. **最新法令・判例レビュー必須**：156条（虚偽公文書作成）・157条（公正証書原本不実記載）・155条（公文書偽造）、
   最判昭51.4.30（写真コピーの文書性）・最判昭51.5.6（補助公務員）・最判昭32.10.4・最判昭27.12.25 を一次確認。

## 3. 現行 v11 の 8記述（内容は妥当・v13 化時の素材）

1. 最判昭51.4.30：複写機による公文書コピーは原本と同一の意識内容を保有し原本同様の社会的機能・信用性 → 公文書偽造罪の客体になり得る（○）
2. コピーの名義人は原本作成名義人と解する（○）
3. 反対説：コピーそれ自体を原本として行使する場合を除けば名義人はコピー作成者（学説・○）
4. 最判昭51.5.6：補助公務員も一定の基本的条件に従う限度で上司名義の公文書作成権限あり（○）
5. 反対説：有形/無形偽造を区別する以上、内容正確なら…（学説・○）
6. 作成権限ある公務員の職務上の虚偽公文書作成＝虚偽公文書作成罪（156条）（○）
7. 最判昭32.10.4：起案担当補助公務員が虚偽起案し情を知らない作成権限者を利用＝虚偽公文書作成罪の間接正犯（○）
8. 最判昭27.12.25：非公務員が虚偽申立てで情を知らない公務員に虚偽公文書を作らせた間接正犯は、特別規定(157条)のある公正証書原本不実記載等に限り可罰（○）
