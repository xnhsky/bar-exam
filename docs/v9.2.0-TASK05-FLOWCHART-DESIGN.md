## 2. ノード仕様

| ノード | 形状 | 色 | サイズ |

| START | 角丸長方形 rx=10 | `--accent-darker` 背景・白文字 | 150×50 |
| Decision diamond | polygon（菱形） | `--accent-light` 背景・白文字 | 180×120 |
| 終端 成立 | 角丸長方形 rx=10 | `--recall-correct` (#1b5e20) 背景・白文字 | 160×50 |
| 終端 不成立 | 角丸長方形 rx=10 | `--contrast-warm` (#d97a4f) 背景・白文字 | 160×50 |
| 肢マーカー | pill rx=11 | `--mid-warm` 背景・白文字 | 60×22 |

Decision diamond の polygon points 計算式：
```
points="(cx-90),(cy) (cx),(cy-60) (cx+90),(cy) (cx),(cy+60)"
```
中心 (cx, cy) を中心に幅 180 × 高さ 120 の菱形。

## 6. viewBox 可変ルール

```python
def select_flowchart_viewbox(num_decisions: int) -> str:
    if num_decisions <= 3:
        return "0 0 900 600"
    elif num_decisions <= 5:
        return "0 0 900 800"
    else:
        return "0 0 900 1000"
```

各 decision diamond の y 配置：
- START: y=40
- Decision i: y = 40 + 90 + (i-1) * 150
- 各 decision diamond の高さ 120、隣接 diamond との垂直間隔 30
- 終端ノード: 最終 decision diamond の y + 150
