## 追加 5 fields

### 1. `spec_version` ← 既存だが値を更新

```json
{ "spec_version": "v9.4.0" }
```

footer-spec feature-tag に `TX v9.4.0 COMPLETE-BASELINE` が出力される。

### 2. `exam_badges[]`（top-level・任意）

```json
{
  "exam_badges": ["📚 刑法", "📝 司法試験", "📅 R6", "🔢 R6-6"]
}
```

未指定時は subject + exam + year + source から自動構築。

**HTML 出力位置**: header 内 `<div class="header-top">` ブロック（`<h1>` の上）。

### 3. `theme_tags[]`（top-level・任意）

```json
{
  "theme_tags": ["横領罪", "背任罪", "共同占有", "親族間特例", "債権譲渡通知前の弁済", "二重抵当", "公務員の任務違背", "判例の立場"]
}
```

論点キーワードタグ N 件。313.html では 8 件出現。

**HTML 出力位置**: header 内 `<h1>` の下、exam-meta の上（`<div class="theme-tags">`）。

### 4. `difficulty`（top-level・任意）

```json
{ "difficulty": "★★☆" }
```

未指定時は `correct_rate` から自動判定：
- correct_rate ≥ 70: `★☆☆`（易）
- 40 ≤ correct_rate < 70: `★★☆`（中）
- correct_rate < 40: `★★★`（難）

**HTML 出力位置**: exam-meta 内 `<span><strong>難度:</strong>★★☆</span>`。

### 5. `choices[*].summary_html`（任意・per choice）

```json
{
  "choices": [
    {
      "label": "ア",
      "summary_html": "共同保管中の財物を独断で持ち出した事案。判例は共同占有者の1人が他の共同占有者の占有を侵害したものとして<span class=\"case-emphasis freq-high\">窃盗罪</span>を成立させており、横領罪は成立しない。"
    }
  ]
}
```

各 choice の 1-2 行要約（raw HTML 可・`case-emphasis` 等の強調タグ使用可）。

**HTML 出力位置**: 各 choice-section 内、choice-header-block の閉じ `</div>` 直前。

```html
<div class="choice-header-block">
  <div class="choice-big-badge">ア</div>
  <span class="verdict ...">○ 正しい</span>
  <div class="choice-summary">共同保管中の財物を…窃盗罪…横領罪は成立しない。</div>
</div>
```

### 6. `choices[*].basis_link_card`（任意・per choice）

```json
{
  "choices": [
    {
      "label": "ア",
      "basis_link_card": {
        "label": "📎 参照する条文・判例",
        "items": [
          {"href": "case-dairen-t11-12-15", "label": "大連判大11.12.15（不動産詐欺既遂時期）", "kind": "case"},
          {"href": "law-246", "label": "刑法246条（詐欺罪）", "kind": "statute"}
        ]
      }
    }
  ]
}
```

各 choice が参照する条文・判例への直接リンク sub-card。

**HTML 出力位置**: 各 choice-section の末尾（professor card の後）。

```html
<section class="choice-section ..." id="choice-1">
  ... (既存 sub-card original / explanation / professor) ...

  <div class="sub-card basis-link">
    <span class="label">📎 参照する条文・判例</span>
    <ul class="lead-list">
      <li><a class="ref-case" href="#case-dairen-t11-12-15">大連判大11.12.15...</a></li>
      <li><a class="ref-stat" href="#law-246">刑法246条（詐欺罪）</a></li>
    </ul>
  </div>
</section>
```

### 7. `mindmap_section`（top-level・任意）

```json
{
  "mindmap_section": {
    "title": "論点詳細マインドマップ",
    "intro_key_phrase_html": "<span class=\"kp-strong\">本問の論点 4 系統</span>を一望する",
    "figure": {
      "svg_html": "<svg viewBox=\"0 0 1100 900\">...</svg>",
      "caption": "図：本問論点の放射状俯瞰"
    },
    "legend": "凡例：中心 = 本問テーマ / 第 1 環 = 論点系統 / 第 2 環 = 判例規範"
  }
}
```

v9.1.0 専用の `<section id="mindmap">` 構造（313 で出力されていた）。

**HTML 出力位置**: basis section の直後・C-1 section の前。post-process 注入。
